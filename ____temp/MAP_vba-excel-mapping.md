## VBA → Excel 列号映射关系

**基础计算：** `位谕列始全部 = 位qt主宽 + 1 = 28`
即：C28 = 第一个位谕of常量。每个位谕of常量值为 `28 + 组起始偏移 + 组内偏移`

### 核心决策链条

```
WXCD(C88) → 金银/屎尿 一票否决
  → WJC(C77) → 周线多空分水岭
    → WJB(C57) → 不破WJB不退
      → DJE(C128) → 日线唯一止损
        → DJC(C106) → 入场参考
          → DJA(C94) → 防震荡
            → DXAB(C89) → 择时入场
```
