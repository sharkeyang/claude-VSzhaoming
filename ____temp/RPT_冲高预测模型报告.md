## 冲高预测模型（推荐规则组合）

### 规则A：高概率买入（冲高预期 > 35%）
```
条件组合（AND）：
  [盈提示 = "高" 或 "a高" 或 "雀高" 或 "a连3高"]
  或
  [今日涨 > +3%] AND [仓月 = A3 或 C7]
  或
  [日周联动 含 "盈"] AND [星期 = 五]
  或
  [大局 = "金升a↗.2栅" 或 "金升a↗.1柜"]
```

### 规则B：中等概率买入（冲高预期 25-35%）
```
条件组合（AND）：
  [今日涨 > +3%]
  或
  [仓月 = A3 或 A1 或 C7]
  或
  [盈提示 = "a连3" 或 "宽高"]
  或
  [日周联动 = 金上b.盈 或 金上r.盈]
```

### 规则C：低概率（不做多）
```
条件组合（AND）：
  [大局首字 = "屎"]
  AND [仓月 = Z0 或 Z4 或 Z6]
  AND [今日涨 < -3%]
```

## 结论

**最有预测力的指标排名：**
1. **今日动量**（+3~7% → 冲高概率翻倍）— 但这是追涨，需谨慎
2. **盈提示中的"高"系列**（宽高/雀高/a高 → 冲高35-57%）— **最佳非动量指标**
3. **仓月=A3升主**（冲高21-41%）— 稳定的趋势信号
4. **日周联动+"盈"后缀**（冲高27-39%）— 日内共振信号
5. **大局=金升** 且子类为 2栅/1柜（冲高40-50%）

**最佳组合信号：**
> **今日涨+3%以上 + 盈提示=高 + 仓月=A3/C7 + 日周联动带盈**
> → 下日冲高概率可达 **50%+**
