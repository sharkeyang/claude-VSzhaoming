#### 2.2.4 DJC之上操作分析流程

**宏观流程：DJE之上 → DJC判定**

```
```

DJE之上 (DXZE>0)
│
├── DJC之上 (DXZC>0)  ← 主要操作区域
│   ├── ZA>0  → 持主动 / 买漏日 ✅
│   ├── ZA≤0  → 持被动 / 买漏周 ✅
│   ├── ZC=1  → 买初介 ✅
│   ├── 下破DJC → 卖止浮 ✅
│   ├── 连3阳  → 卖止盈 ✅
│   └── 上破DJA≤3日 → 卖诱多 ✅
│
└── DJC之下、DJE之上 (DXZC≤0, 但 DXZE>0)
    ├── CD>0, ZC<0, 护级已破损
    ├── CD<0, 趋势走弱
    └── 此区域只持基仓，不操作浮仓

```

**操作流：**
1. 判定 `DXZE>0`（在DJE之上）→ 进入浮仓操作区域
2. 进一步判定 `DXZC`：
   - `DXZC>0`（DJC之上）→ 主要操作区域，按具体信号执行
   - `DXZC≤0` 但 `DXZE>0`（DJC之下过渡区）→ 只持基仓，不操作浮仓
3. 在DJC之上区域内，根据条件触发相应操作（买入/持有/卖出/退出）
---
**微观决策树：DJC之上操作分类**

```
```

DJC之上（DXZC>0）
│
├── [退出] 下破DJC → 浮仓退出
│
└── [持有] DJC之上
    │
    ├── 提前退出信号（非跌排，但应提前退出）
    │   ├── 止盈（偏离大、连阳等）          → 标注 [卖止盈]
    │   └── 诱多（上破DJA三柱内/连阴后阳/阴阳阴后阳） → 标注 [卖诱多]
    │
    ├── 介入信号
    │   ├── 上破DJC（与退出对应）            → 买初介
    │   └── 中途介入（买捡漏）
    │       ├── DJA之上日级别捡漏             → 买漏日
    │       └── DJA之下周级别捡漏（基于周柱排）→ 买漏周
    │
    └── 持有条件（无条件持有）
        ├── DJA之上 → 持主动，直到出现提前退出信号或下破DJC
        └── DJA之下 → 持被动（除非是<买漏周>机会）

```
