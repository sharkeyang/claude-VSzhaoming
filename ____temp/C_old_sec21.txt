### 2.1 专题一：多长仓位与冲高策略的统一（2026-06-30）

**致力于解决问题：** _多长区是否需要区分"基仓"和"浮仓"？<周浮仓>/<日浮仓>与浮仓的关系是什么？

**讨论结果：**

#### 2.1.1 四区划总览（从好→差）

**分区解释：** _多长与\_多被合起来是 `WXZC>0 且 DXZE>0` 的区域，即 **WJC必赢区域**。<_多长>与<\_多被>是将此区域划分为四类：

| 子区域 | WXCD | WXAB | 归属 | 说明 |
|:------|:----:|:----:|:----|:-----|
| ① 唏 | 唏 | 不限 | **\_多被（试仓）** | 试仓区域，成功率低 |
| ② 银 + 不限 | 银 | 不限 | **\_多被（试仓）** | 银的WXCD≤0但WXZD>0，归入多被仅小仓位试错 |
| ③ **金 + 甲乙己** | **金** | **甲/乙/己** | **_多长（主升）** | 依赖WXAB护线的主升区域 |
| ④ 金 + 丙丁戊 | 金 | 丙/丁/戊 | **\_多被（被动）** | 被动持仓区域，要么下破WJC/DJE止损，要么重新变为_多长 |

<span style=color:red>> **警示二：** 每当想在 `<_空看>` 或 `<_空长>` 介入时，就思考在 WJC 之下介入的 **盈利损失比**，明确不要"丢西瓜捡芝麻"。</span>

<span style=color:red>> **警示五：** `<银>` 被排除 `_多长`，是因为根据实际观察，当 WXCD<0 且 WXZD>0 时，很容易满足 WXZB>0（即 WXAB 甲乙己），但在 WXCD<0 的区域，属于过渡区域，很容易出现下破 WJB 导致趋势终结，而且是无预兆地终结（因为主力在建仓，必须造成恐慌）。</span>

#### 2.1.2 _多长策略 = 单一基仓

<table border="1" cellpadding="8" cellspacing="0" style="border-collapse:collapse;border:2px solid #888;margin:8px 0">
<tr><td>

_多长区的本质是"金+甲乙己"的优质趋势行情。在这种行情中，策略应保持单一：

| | 操作 | 条件 |
|:-|:----|:-----|
| **进入** | 持有基仓 | 区划判定为_多长 |
| **退出** | 清仓 | 下破DJE或下破WJC |

**<长基仓策略>与<_多长>走势相关：**

| 条件 | 操作 |
|:-----|:----|
| 🟢 **优先选择**：WXAB刚正交 | 主升概率高，可介入 |
| 🔴 **避免选择**：虽为_多长但不宜介入的情况 | 等待或放弃 |
| &nbsp;&nbsp;· "最后一柱"形态 | 主升末端，追高风险大 |
| &nbsp;&nbsp;· 失势升排 | 升势衰竭，继续上升空间有限 |
| &nbsp;&nbsp;· WXZA贯穿诱多 | 短期拉升实为诱多，避免高位站岗 |

<span style=color:red>> **警示一：** 不需要在`<_多长>`区内部分"基仓/浮仓"——_多长就是基仓，持有了就不要频繁进出。</span>

<span style=color:red>> **警示二：** `<_多长>`策略就是依赖EMA数学原理，底层是（WXCD正交+WXAB正交），所以不要思考止盈、提前退出等问题。深刻理解"大道至简"的精髓。</span>

</td></tr>
</table>


#### 2.1.3 冲高策略 = 浮仓（本质统一）


<周浮仓>、<日浮仓>、浮仓，三者是同一件事的不同时间尺度表现。不同策略存在三种差异。

<table border="1" cellpadding="8" cellspacing="0" style="border-collapse:collapse;border:2px solid #888;margin:8px 0">
<tr><td>

**差异一：操作时点维度的差异**

| 策略 | 操作时点 |
|:----|:---------|
| **<日浮仓>** | 依赖日级别走势，前日尾盘买入，次日冲高时寻机卖出 |
| **<周浮仓>（澄清）** | 只要依赖于周级别指标进行浮仓操作，都属于<周浮仓>。操作时点从前周五到本周四每一天尾盘均可操作 |

<周浮仓>具体操作时点选择：
- 🟢 **优选前周五尾盘买入** — 可以吃到源于无序震荡的整周冲高幅度
- 🟡 **如预感下周初会冲低** — 可等待下周初冲低后再介入
- 🔴 **必须在周五收盘前卖出**

**差异二：走势介入时点的差异**

浮仓只在`_多长`趋势的中途进行操作。

`_多长基仓`则只针对WXZC，从始至终持有。

**差异三：走势的差异**

<日浮仓>**只应针对DXAB>0且DXZA>0的区域。** 在DJA之下或刚上破DJA时，不应使用<日浮仓>逻辑，而应**依赖周级别信息**（周柱排列、周波型）。

<周浮仓>是依赖周级别信息进行浮仓冲高的**一个特例**，针对**DXZA<0或DXAB<0的情况**。它限制了入仓时间（周五尾盘）和出仓时间（下周内）。本质与广义浮仓一致。

| 策略 | 本质 | 级别 | 介入点 | 退出点 |
|:----|:-----|:----|:-------|:-------|
| **浮仓（广义）** | 浮仓 | 不限 | 条件触发时 | 条件消失时 |
| **<日浮仓>** | 浮仓—日频 | 日线级别 | DJA之上，DXZA>=0，DXAB>0 | 次日或达标 |
| **<周浮仓>** | 浮仓—周频 | 周线级别 | 周柱排列+周波型，DXZA<0或DXAB<0 | 下周内 |

三者逻辑相同：**在`_多长`的保护下，用浮仓做短线脉冲**。
</td></tr>
</table>

#### 2.1.4 对VBA代码的影响

当前 `IQQQ跨码_D据擎2神谕.bas` 中`_多长`区旧逻辑将基仓/浮仓混在同一个区域内，需重构为独立分层：

```
旧逻辑：
  _多长区 → 基仓(BTZA>0) / 浮仓(BTZA≤0) / 减仓(BTZC≤0) / 清仓(BTZE≤0)

新逻辑：
  _多长区 → 仅基仓（单一仓位，满足条件即持有，不下破DJE不退）
  浮仓   → 独立操作模式，由冲高信号触发，不隶属_多长区
              ├── <周浮仓>：基于周柱排/周波型，DXZA<0或DXAB<0时启用
              └── <日浮仓>：基于日柱排，DXAB>0且DXZA>0时启用

  仓位跟随区划与信号：
    _多长      → 持有基仓
    _多被   → 可参与（视情况开基仓）
    _空看    → 不参与
    _空长   → 不参与
    浮仓信号触发 → 在_多长保护下开浮仓
```

**分层策略决策树（VBA实现）：**

```
                  ┌─ <长基仓> ──→ 长期策略（针对_多长，按WXZC>0持有）
                  │
当前持仓 ──────────┤
                  │          ┌─ 能<日浮仓>？→ <日浮仓>策略
                  └─ 浮仓 ──┤
                            └─ 不能？→ 能<周浮仓>？→ <周浮仓>策略
                                      │
                                      └─ 都不能？→ 平浮仓，回归基仓
```

**验证闭环：** M3回测给出了"理论上三层策略是否有效"的定量答案；M5实时提示给出了"当前持仓应该怎么做"的操作指令。

**待代码实现（优先级排序）：**
1. [ ] 重构_多长区仓位逻辑：移除_多长浮仓/_多长减仓，统一为_多长基仓
2. [ ] 浮仓逻辑独立：冲高信号（下柱冲高提示/日柱排条件）直接触发浮仓入场，不再作为_多长区的子分类
3. [ ] 仓位退出统一由区划切换或信号触发（区划≠_多长时清仓基仓，下破DJC时平浮仓）
