# MC3 策传映射表 — 策传数据源追溯 + 平衡策分卡

## 设计思路

策传需要**合流两套数据**：

**① 结构化推荐卡（8项）** — 用于每日持仓监控与决策参考
**② 走势问题查询表** — 用于标准化决策流程（见§⑤）

实施策略：**VBA生成原始数据（Z sheet策传列），Python格式化为最终策传**。

---

## ① 策传映射表（决定性+参考性）

因为只有**四域**具有决定性，所以在策传里：
- **1️⃣ 一票否决、2️⃣ 四域**是**决定性**的 — 四域决定能否操作
- **3️⃣ 日级别、4️⃣ 综合参考**是**参考性**的 — 辅助判断时机
- **5️⃣ 走势策分**是**参考性**的 — 信心指数
- **6️⃣ 策略**依据1~4制定，但只有 `<长基仓>` 具有确定性，`<周浮仓>`次之，`<日浮仓>`最差
- **7️⃣ 仓位**根据策略确定
- **8️⃣ 后续走势预案**根据策略、仓位与后续走势确定

| # | 项目 | VBA变量名 | 取值范围/逻辑 |
|:-:|:-----|:---------|:-------------|
| 1️⃣ | **一票否决**（决定性） | | 条件：前周WXZC>0 **且** 今日DXZE>0 **且** WXCD=金/银 |
| | **WXZC>0**（前周） | `位谕of日类BTZC` | > 0（日线JZ上JC） |
| | **DXZE>0**（今日） | `位谕of日类BTZE` | > 0（日线JZ上JE） |
| | **WXCD=金/银** | `位谕of周层大局` | `Mid$(值WXCD护型, 2, 1)` = 金/银 |
| 2️⃣ | **四域**（决定性） | | |
| | **周四域** | `位谕of周层四域` | 多长✅ / 多被⚠️ / 空看❌ / 空长❌ |
| | **WXAB** | `位谕of周层护型` | 甲乙丙丁戊己；含甲乙己 ✅，否则⚠️ |
| | **WTAB** | `位谕of周类BTAB` | 刚交(≤3)/正交(3~10)/久交(>10)；若WTZA≤3 → ⚠️诱多 |
| 3️⃣ | **日级别**（参考性） | | |
| | **仓日类** | `位谕of仓日类` | 无→⚠️（DJE之下）；否则`[CD方向][AB等级]`：上a✅上b✅上c⚠️下*❌ |
| | **DXCD** | `位谕of日层联动`（首字） | 金/银/唏/嘘/屎/尿 |
| | **DXAB** | `位谕of日层护型` | 甲乙丙丁戊己 |
| | **日层段** | `位谕of日层段` | 买初/持主/持被/卖浮（互斥） |
| | **日层信号** | `位谕of日层机警` | 警盈/警诱/漏日/漏周（可重叠） |
| 4️⃣ | **综合参考**（参考性） | | |
| | **周波形** | `位谕of周层波型` | 龙猪/龙管→趋势强劲⚠️注意止盈；头正→头部形成⚠️关注转折；头负→弱势❌规避 |
| | **周柱排** | `位谕of周层柱排` | 升.→方向向上；跌.→方向向下；(人)→方向不明，人工判定 |
| | **日波形** | `位谕of日层波型` | 龙猪/龙管→短线强势⚠️注意止盈；震正→震荡等待；头负/震负→短线弱势❌ |
| | **日柱排** | `位谕of日层柱排` | 升→短线向上注意止盈；跌→短线向下❌非介入点；人→方向不明 |
| | **盈提示** | `位谕of日层盈提示` + `位谕of周层盈提示` | 高→涨高了⚠️是否止盈；宽→波动大⚠️是否减仓；丘→丘幅预警 |
| | **机阱提示** | `位谕of周波龄BTPR0` + `位谕of周波今涨幅` | 仅四域=多长时：升连≥3→过热⚠️注意顶尾；跌连≥2→回调✅关注<周浮仓>机会；阱涨(≥20%)→⚠️止盈；阱跌(≤-11%)→❌止损 |
| 5️⃣ | **走势策分**（参考性） | 平衡策分卡 | 四域(50) + WXAB+WTAB(10) + 日级别(20) + 综合参考(20) |
| 6️⃣ | **策略** | | <长基仓>确定性最高，<周浮仓>次之，<日浮仓>最差 |
| | **<长基仓>** | `四域周` ∈ {多长,多被} | 多长时介入，多被时继续持有 |
| | **<周浮仓>** | `下柱冲高提示`≠"" + `ZA周`>0 + `ZC日`>0 | 介入：多长+前周WTZA>0+ZC日>0 |
| | **<日浮仓>** | `仓日类`含"上" + `ZA日`>0 + `柱排`首字=升 | 介入：多长+前周WTZA>0+DXAB>0+DTZA>0 |
| 7️⃣ | **仓位** | 来自持仓表（Excel） | 长基仓最高单股10000；浮仓最高单股20000；尾盘一次买够，分批卖出 |
| 8️⃣ | **后续走势预案** | `日层段` + `日层机警` | 规则引擎匹配：日层段+日层信号驱动 |

## ④ 平衡策分卡（百分制）

**结构：** 一票否决（门槛）→ 四域(50) → WXAB+WTAB(10) → 日级别(20) → 综合参考(20) = 100分

> ⚠️ **使用注意：** 策分的主要价值在于**区分层级**（多长 vs 多被 vs 空看 vs 空长），**同一层级内的分数高低参考性有限**。例如多长内部的66分和70分对后续走势影响不大。详见回测验证结论。

### 一票否决（门槛）
| 条件 | 结果 |
|:----|:----:|
| WXZC>0 AND DXZE>0 AND WXCD=金/银 | 通过，进入评分 |
| 任一不满足 | 0分，直接淘汰 ❌ |

### 四域（满分50分）
| 条件 | 得分 | 说明 |
|:----|:----:|:-----|
| 周四域=多长 | 50 | 最佳状态，可积极操作 |
| 周四域=多长 + WTZA≤3（诱多） | 40 | 诱多风险-10，警惕回调 |
| 周四域=多被 | 25 | 可参与但不够好 |
| 周四域=空看 | 10 | 不推荐主动介入 |
| 周四域=空长 | 0 | 禁止参与 |

### WXAB + WTAB（满分10分）
| 条件 | 得分 | 说明 |
|:----|:----:|:-----|
| WXAB=甲/乙/己 | +5 | 护型合格 |
| WXAB=丙/丁/戊 | 0 | 护型不合格 |
| WTAB=正交（3~10日） | +5 | AB交叉稳定 |
| WTAB=刚交（≤3日） | +3 | AB刚交叉，待确认 |
| WTAB=久交（>10日） | +2 | AB长期交叉，效力减弱 |

### 日级别（满分20分，有加有减）
| 条件 | 得分 | 说明 |
|:----|:----:|:-----|
| **仓日类** | | |
| 上a / 上b（CD向上+AB优良） | +10 | DJE之上+CD向上+AB甲/乙 |
| 上c / 其他（CD向上+AB一般） | +5 | DJE之上但AB为丙/丁/戊 |
| 无（DJE之下） | -10 | DJE之下，需⚠️警示 |
| **日层段** | | |
| 持主（主动持有） | +5 | DJA之上无卖出信号 |
| 买初（刚上破DJC） | +3 | 刚浮仓入场 |
| 持被（被动持有） | +2 | DJA之下可参与 |
| 卖浮（需退出） | -5 | 下破DJC应退出 |
| **日层信号** | | |
| 有漏日/漏周（机会信号） | +5 | 有捡漏机会 |
| 有警诱（诱多预警） | -3 | 诱多风险 |

### 综合参考（满分20分，有加有减）
| 条件 | 得分 | 说明 |
|:----|:----:|:-----|
| **周波形** | | |
| 龙猪/龙管（主升趋势） | +5 | 周级别趋势强劲 |
| 头正（头部形成中） | +2 | 注意冲高 |
| 震正（震荡整理） | 0 | 方向不明 |
| 头负/震负（弱势） | -5 | 周级别走弱 |
| **日波形** | | |
| 龙猪/龙管（主升趋势） | +5 | 日级别趋势强劲 |
| 头正（头部形成中） | +2 | 注意冲高 |
| 震正（震荡整理） | 0 | 方向不明 |
| 头负/震负（弱势） | -5 | 日级别走弱 |
| **冲高信号** | | |
| 有下柱冲高提示（周浮仓） | +5 | 周级别浮仓机会 |
| 有漏提示（日浮仓） | +3 | 日级别浮仓机会 |
| 无冲高信号 | 0 | — |
| **柱排方向** | | |
| 日柱排=升 | +3 | 日级别方向向上 |
| 日柱排=跌 | -2 | 日级别方向向下 |
| 日柱排=人 | 0 | 方向不明 |
| **机阱风险** | | |
| 升连≥3周（过热） | -5 | 连涨多周，注意回调 |
| 跌连≥2周（回调机会） | +3 | 回调后可关注介入 |

### 总分判定
| 总分 | 评级 | 操作建议 |
|:---:|:----|:---------|
| **≥80分** | 强烈推荐 ✅ | 多维度优秀，可积极操作 |
| **60~79分** | 可参与 ⚠️ | 核心指标通过，部分维度需关注 |
| **<60分** | 观望 ❌ | 过多扣分项，不急于介入 |
| **一票否决** | 淘汰 ❌❌ | 核心条件不满足，直接放弃 |

### 回测验证结论（10只股票37703行数据）

**策分卡的角色：质量展示，不是决策依据。**

| 验证项目 | 结论 | 说明 |
|:--------|:-----|:------|
| 多长 vs 多被 | 天差地别 | 多长20日胜率**57.9%**，多被**50.5%** |
| 多长内部分数区分度 | 无法区分 | 76%的样本挤在66-70分区间 |
| 加分项预测力 | 微弱 | WXAB/WTAB/日层段在多长内部影响有限 |
| 次日涨跌预测 | 不可行 | 所有分数段次日胜率≈50% |

**核心结论：持仓的真正依据是周级别四域=多长，而不是短期指标决定的策分。**

```
长基仓的决策逻辑：
  一票否决 → 周四域=多长? → 持有（长基仓策略）
               ↕
   策分卡分数仅作为"信心指数"展示：
     ≥70 = 多长中各方面条件优秀
     60~69 = 多长中条件一般
     <60 = 多半是因为多被，应等待进入多长
```

## ⑤ 走势问题查询表（持续更新 — SOP流程）

**目标：** 形成标准化操作流程（SOP）——遇到什么走势情况→查看什么指标→各指标的影响力→应该进行什么操作。

**应用场景：** 每日持仓监控 + 备选推荐审核。

| 走势情境 | 需问的问题 | 查询指标 | 注意 | 解决方案 |
|:---------|:-----------|:---------|:----|:---------|
| **四域=多长** | 趋势能否持续？止损位上移了吗？ | 四域周=多长, WXAB=甲/乙, WTAB≤3? | 🟢 持有 | DJE止损位随ZA上移，ZA>0则持有 |
| **四域=多被** | 何时转强？加仓条件？ | WXAB是否从丙丁戊→甲乙, ZC周>0? | 🟡 等待 | 不加仓，等WXAB转甲/乙+ZC周>0 |
| **四域=空看** | 是否已减仓？反转信号？ | DXZE>0? ZC周拐头？仓日类≠无? | 🟠 警惕 | 已减仓则等四域转多，未减仓则减 |
| **四域=空长** | 为何持有？止损触发？ | ZE日<0? 仓日类=无? 肉垫亏损>10%? | 🔴 危险 | 清仓或减仓，等四域转多再考虑 |
| **日段=持主** | 止盈目标在哪？顶部信号？ | 波形=头正/头负? 冲高=有信号? | 🟢 持有 | 波形头正→准备止盈，ZA>0则继续持 |
| **日段=持被** | 减仓线定在哪？补仓吗？ | ZA日<0? DJE支撑? 肉垫亏损幅度 | 🟡 观察 | 不补仓，破DJE则减仓，等ZA转正 |
| **日段=卖浮** | 浮仓已卖？基仓同步减？ | ZC下破? DJE支撑? 仓日类? | 🟠 执行 | 浮仓平仓，基仓视四域决定 |
| **临涨>8%** | 是否见好就收？冲高回落？ | 猴市策略, 成交量放大? 波形顶? | 🔥 注意 | 分批止盈，不贪 |
| **临涨<-5%** | 止损触发？大盘环境？ | 大盘WXCD, 板块轮动状态 | 🔥 注意 | 触发止损则执行，未触发则观察 |
| **仓日=无** | DJE之下为何持有？有无均线支撑？ | ZE日<0, ZD<0, ZC<0 | 🔴 危险 | 无均线支撑的持仓需质疑 |
| **仓日=忐/忠/忑** | 临界态方向选择？ | 次日柱排方向, DXAB方向 | 🟡 观察 | 方向确认前不宜重仓 |
| **盈提示=高** | 是否涨高了？止盈？ | 周龄, 连涨次数 | 🟠 注意 | 周龄>3+连涨→考虑止盈 |
| **漏周/漏日** | 捡漏机会？反转信号？ | 四域是否转多, ZC>0? | 🟢 关注 | 四域多长+漏周=可关注周浮仓 |
| **波型=龙猪/龙管** | 主升中，是否接近顶部？ | 盈提示, #H10频次, 成交量 | 🟢 持有 | 趋势跟随，注意止盈信号 |
| **波型=头正** | 头部形成，还能再冲？ | 柱排方向, 冲高提示 | 🟠 警惕 | 准备减仓，等跌破确认 |
| **波型=震正** | 震荡等待方向 | 柱排方向, DXAB方向 | 🟡 观察 | 方向确认前不加仓 |
| **波型=头负/震负** | 弱势阶段，规避 | 四域是否转空, 仓日类 | 🔴 危险 | 减仓或清仓 |

**更新规则：** 每次复盘发现新的有效/无效信号就追加一行。验证方法见§⑥复盘回放。

## ⑥ 复盘回放方案

### 目标
验证每个策传信号在历史数据中的有效性：信号出现后1/3/5/10天的后续表现。

### 数据采集
每日收盘后记录Z sheet + AKShare数据，关键字段：

| 字段 | 来源 | 说明 |
|:-----|:-----|:-----|
| 日期 | 自动 | YYYY-MM-DD |
| 股票代码 | Z sheet Col1 | 全市场标的 |
| 四域周 | Z sheet Col308 | 多长/多被/空看/空长 |
| 仓日类 | 策传解析 | 上a/上b/无/忐b等 |
| 日段 | 策传解析 | 持主/持被/卖浮/空 |
| 策分 | 由四域+日段+仓日计算 | 0-100 |
| 策传全文 | Z sheet Col315 | 5个积木块原文 |
| 后续1/3/5/10日涨幅 | AKShare | 验证信号有效性 |

### 验证方法
对每个信号统计胜率：`信号A出现 → 后续N天上涨概率 = 上涨次数 / 总出现次数`

| 验证问题 | 分组条件 | 成功标准 |
|:---------|:---------|:---------|
| 四域=多长是否有效？ | 多长vs多被vs空看vs空长 | 多长5日胜率>55% |
| 日段=持主是否有效？ | 持主vs持被vs卖浮 | 持主3日胜率>55% |
| 仓日类=无是否应止损？ | 无vs上a vs 上b | 无=后续5日胜率<45% |
| 漏周信号是否有效？ | 有漏周vs无漏周 | 漏周后3日胜率>55% |
| 策分区分度 | ≥70 vs 60-69 vs <60 | 高分组合后续胜率更高 |
| 临涨信号是否有效？ | 临涨>8% vs >5% vs <0 | 临涨>8%后3日回落概率>60% |

### 实施步骤
1. **Phase 1（数据积累）**：每天运行采集脚本，将Z sheet + AKShare数据写入CSV/DB，连续30天
2. **Phase 2（回测分析）**：30天后对积累数据做统计，计算每个信号的胜率
3. **Phase 3（规则优化）**：根据回测结果，调整走势问题查询表的注意程度和解决方案
4. **Phase 4（持续迭代）**：每月更新一次回测结果，信号有效性衰减的降级，新增的补充
