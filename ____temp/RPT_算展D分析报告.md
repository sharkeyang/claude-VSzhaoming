## 三、 T@乾坤策略核心规则 (来自PX算研_RAP算展9算灌.bas)

### 周类开仓条件
```
规则周类 = (周类BTZC > 0 And 周类BTZB > 0)
```
即: 周类价格在均线C之上 AND 均线B在均线C之上 (均线多头排列)

### 日类开仓条件
```
规则日类 = (日类BTZE > 0 And 日类BTZD > 0)
```
即: 日类均线E在均线F之上 AND 均线D在均线E之上

### 仓位规则
- 条件满足 → 满仓(1单位)
- 条件不满足 → 空仓(0)
- 每次开仓投入 10000 元基准

### 均线系统 (JZ=基准, JA-JG=7条均线, 周期递增)
```
BTZA = JZ >= JA (价格在A均线上)
BTZB = JZ >= JB
BTZC = JZ >= JC
BTZD = JZ >= JD
BTZE = JZ >= JE
...
```

## 四、 策略收益验证 — 结论：**不稳定，不推荐原样使用**

### 4.2 日类策略 (D) — 微利但胜率极低

### 4.3 关键发现问题
1. **胜率极低**: 日类仅20.5%, 周类23.4%, 即80%的交易是亏损的
2. **盈利靠极少数大单**: 日类27笔盈利 VS 105笔亏损, 全靠少数大赚撑着
3. **周类净亏损**: 27.8年周期最终亏钱
4. **日类年化收益极低**: 10651 ÷ 27.8年 ≈ 383元/年, 远跑输无风险利率
5. **信号过于简单**: 仅靠2个均线交叉条件, 不考虑量价关系、市场状态

## 七、 重要发现与建议

### 1. T@乾坤策略不可直接用于实盘
- 27.8年回测胜率仅20%, 周类亏损
- 信号过于简单(仅2个均线条件)
- 建议: 结合仓月/大局/护型等更丰富特征做决策

### 2. 最有价值的预测因子排名
1. **仓月** → 下周预测最强(A3→+5.51%)
2. **大局首字** → 市场状态分类
3. **护型WXAB** → 趋势方向与力度
4. **动量(当前涨跌)** → 短期反转/延续
5. **漏提示** → 含丰富模式识别信息

### 3. 隐藏列的价值
大量隐藏列(如仓月col230、漏提示col93、大局col88等)含有高预测力信息, 建模时必须纳入

### 4. 具体信号示例
- A3升主+金升+护型a甲↗上 → 高概率持股
- C7顶震+屎降+护型c丙↘下 → 高概率卖出
- 大跌-9%以上 → 短期反弹概率80%
- 中跌-5~-6% → 继续下跌概率73%
