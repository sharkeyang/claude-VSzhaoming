**日期：** 2026-06-29

## 一、单柱分类（并符基元）

### 单根K线的并符判定

| PR0(本周期涨跌) | PR1(上周期涨跌) | C0 vs O1 | 并符 | 符号 | 含义 |
|:---------------|:---------------|:---------|:----|:----:|:-----|
| <0 (跌) | <0 (上期也跌) | — | **跌连** | `W` | 连跌势头 |
| <0 (跌) | ≥0 (上期涨) | C0<O1 (收在开盘下) | **跌吞** | `V` | 阴包阳 |
| <0 (跌) | ≥0 (上期涨) | C0≥O1 (收在开盘上) | **跌孕** | `v` | 阴孕阳 |
| ≥0 (涨) | ≥0 (上期也涨) | — | **升连** | `Q` | 连涨势头 |
| ≥0 (涨) | <0 (上期跌) | C0≥O1 (收在开盘上) | **升吞** | `O` | 阳包阴 |
| ≥0 (涨) | <0 (上期跌) | C0<O1 (收在开盘下) | **升孕** | `o` | 阳孕阴 |

### 数据来源

```vb
' IQQQ跨码_D据擎5结算.bas:734-751
值单形并符 = ""
If ARROS(位os结幅PR0) < 0 Then                    ' 本周期下跌
    If ARROS(位os结幅PR1) < 0 Then                    ' 上周期也下跌
        值单形并符 = 常枚并符跌连   ' W
    ElseIf ARROS(位os价C0) < ARROS(位os价O1) Then    ' 收盘价<上期开盘价
        值单形并符 = 常枚并符跌吞   ' V
    Else
        值单形并符 = 常枚并符跌孕   ' v
    End If
Else                                               ' 本周期上涨
    If ARROS(位os结幅PR1) >= 0 Then                   ' 上周期也上涨
        值单形并符 = 常枚并符升连   ' Q
    ElseIf ARROS(位os价C0) >= ARROS(位os价O1) Then   ' 收盘价>=上期开盘价
        值单形并符 = 常枚并符升吞   ' O
    Else
        值单形并符 = 常枚并符升孕   ' o
    End If
End If
```

> **术语等价名：** `W` = 跌连 = 连阴 = 连续阴线 | `Q` = 升连 = 连阳 = 连涨 | 文档后续使用"W(连阴/跌连)"统称。

## 二、柱排字符串结构

### 基本格式

```
{方向}.{尾部形态}{符号序列}.{最后并符}{连数}
```

| 段 | 说明 | 示例 |
|:--|:-----|:-----|
| `{方向}` | 升 / 跌 / (升)人 / (跌)人 / (人)人 | 决定主要方向判断 |
| `{尾部形态}` | 尾连 / 尾吞 / 尾反孕 / 连后吞 / 吞吞 / 孕孕 | 最近两根K线的组合形态 |
| `{符号序列}` | 由 `Q O o W V v` 组成的序列 | 用于追溯详细K线形态 |
| `{最后并符}{连数}` | 最后单柱的并符+连续次数(绝对值) | 用于判断尾声 |

### 符号对应表

| 符号 | VBA常量 | 含义 |
|:----:|:--------|:-----|
| `Q` | `常枚并符升连` | 升连（连续阳线） |
| `O` | `常枚并符升吞` | 升吞（阳包阴） |
| `o` | `常枚并符升孕` | 升孕（阳孕阴） |
| `W` | `常枚并符跌连` | 跌连（连续阴线） |
| `V` | `常枚并符跌吞` | 跌吞（阴包阳） |
| `v` | `常枚并符跌孕` | 跌孕（阴孕阳） |

## 三、全部柱排类型及信号映射

### 3.1 升链（方向=升，可直接操作）

| 柱排内容 | 尾部形态 | 信号强度 | 含义 | 操作建议 |
|:---------|:--------|:-------:|:-----|:---------|
| `升.尾连QQ` | 连升 | 🟢🟢🟢 | 连续阳线，升势强劲 | 持有/加仓 |
| `升.尾(连\|吞)[ovVv]+Q` <br>例：`升.尾连Ov`、`升.尾连oQ` | 连升(含孕/吞) | 🟢🟢 | 升链延续，但中途有停顿 | 持有 |
| `升.尾吞vO` | 吞没 | 🟢🟢 | 先跌后涨，吞没前阴 | 买入信号 ✅ |
| `升.尾反孕Qv` | 反孕 | 🟢 | 升链后出现跌孕，可能转弱 | 持有但关注 |
| `升.尾反孕vOv` | 反孕(多次) | 🟡 | 升链中出现反复孕线 | 谨慎持有 |

### 3.2 跌链（方向=跌，不宜买入）

| 柱排内容 | 尾部形态 | 信号强度 | 含义 | 操作建议 |
|:---------|:--------|:-------:|:-----|:---------|
| `跌.尾连WW` | 连跌 | 🔴🔴🔴 | 连续阴线，跌势强劲 | 卖出/不买入 |
| `跌.尾(连\|吞)[oOvV]+W` <br>例：`跌.尾连Vv`、`跌.尾连oW` | 连跌(含孕/吞) | 🔴🔴 | 跌链延续 | 不买入 |
| `跌.尾吞oV` | 吞没 | 🔴🔴 | 先涨后跌，阴包阳 | 卖出信号 |
| `跌.尾反孕Wo` | 反孕 | 🟡 | 跌链后出现升孕，可能转强 | 观察买入点 |
| `跌.尾反孕oVo` | 反孕(多次) | 🟡 | 跌链中出现反复孕线 | 关注反转 |

### 3.3 人排（需人工判断，策略中用均线辅助）

| 柱排内容 | 尾部形态 | 信号 | 含义 | 辅助判断方式 |
|:---------|:--------|:----:|:-----|:------------|
| `(升)人.连后吞Qv` | 连后吞 | 🟡 | 升链后被阴吞，但方向暂判升 | 看WXCD/DXCD是否为正 |
| `(升)人.连后吞QvQ` | 连后吞反转 | 🟢 | 升链被吞后立刻收回 | 买入信号 |
| `(跌)人.连后吞Wo` | 连后吞 | 🟡 | 跌链后被阳吞，方向暂判跌 | 看WXCD/DXCD是否转正 |
| `(跌)人.连后吞WoW` | 连后吞反转 | 🔴 | 跌链被吞后继续跌 | 卖出信号 |
| `(升)人.吞吞vOV` | 吞吞 | ⚪ | 巨幅上下波动，无方向 | 放弃，不清仓也不加仓 |
| `(跌)人.吞吞oVO` | 吞吞 | ⚪ | 巨幅上下波动，无方向 | 放弃，不清仓也不加仓 |
| `(人)人.吞吞* OV` | 吞吞 | ⚪ | 巨幅上下波动，完全无方向 | 放弃操作 |
| `(升)人.孕孕vv` | 孕孕 | ⚪ | 小幅波动，方向待定 | 等待方向明朗 |
| `(跌)人.孕孕ov` | 孕孕 | ⚪ | 小幅波动，方向待定 | 等待方向明朗 |
| `(人)人.孕孕vv` | 孕孕 | ⚪ | 极小波动，无方向 | 等待 |

### 3.4 异常

| 柱排内容 | 含义 |
|:---------|:-----|
| `错.未赋值` | 未匹配到任何已知形态（极少出现） |
>
> **排查指引：**
> ① 检查该股票当日是否停牌（成交量=0 或 无当日数据）
> ② 检查结算周期：周柱排 / 日柱排是否混用（周用日数据会错）
> ③ 检查是否为新上市股票（不足5根K线无法合成柱排）
> ④ 检查数据源是否有跳空缺口或除权除息日
> ⑤ 若连续3期出现"错"，标记该标的为"数据异常"并移出候选池

## 四、尾部后缀：连数指示

柱排末尾会附加 `.Q{N}` 或 `.W{N}` 表示最后单柱的连续次数：

```vb
' 据擎2:1729-1732
If Right(并符串, 1) = 常枚并符跌连 Then    ' 最后一个是W(跌连)
    柱排 = 柱排 & ".W" & (-1 * BTPR0)      ' 加负数连数
ElseIf Right(并符串, 1) = 常枚并符升连 Then ' 最后一个是Q(升连)
    柱排 = 柱排 & ".Q" & BTPR0             ' 加正数连数
End If
```

| 后缀示例 | 含义 |
|:---------|:-----|
| `.Q3` | 最后单柱为阳柱，连涨3天 |
| `.W2` | 最后单柱为阴柱，连跌2天 |

**策略应用：**
- 下日冲高：日柱排后缀 `.Q≥1` 即可（最后是升连，说明涨势延续）
- 下周冲高：周柱排后缀 `.Q≥1` 即可
- 日线止跌（配合 D表 等级判断，见 B §二(7)）：
  - `V0忑`~`V4忑` 阶段出现 `.W{N}` → 仍在空头深处，正常现象，**不视为买入信号**
  - `V7上`~`V9下` 阶段出现 `.W{N}` → DJE上方的巩固回踩，属坐公交范畴，可视为加仓窗口
  - 空头止损：柱排 `.W{N}` + DJE 破位同时出现 → 卖出信号增强至 **-3**

## 五、柱排在策略中的直接应用

### 下日冲高策略（§八 策略三）

```
入场条件：日柱排 以"升"开头 → 确定方向为升链
         AND 尾部不是"反孕" (反孕表示升势可能结束)
         AND 后缀为.Q1以上（最后是升连）
         → 可考虑开仓
```

### 镜面对称❸（DJA阴柱不转跌）

```
持有条件：日柱排 不以"跌"开头         → 不是跌链，忽略阴柱
         AND 日柱排 不包含"W"(连阴/跌连) → 不是连阴/阴阳阴
         AND 日柱排 不是"(跌)人"开头   → 人工判断方向也不是跌
         → 基仓+浮仓继续持有
```

### 浮仓开仓（DJA上方阴柱）

```
开仓条件：日柱排 = "升.尾吞" 开头      → 升吞是标准买入形态
         或 日柱排 = "升.尾连" 开头     → 升连可以追加
         或 日柱排 = "(X)人.连后吞" 且均线为正 → 连后吞需确认
```

### 下周冲高策略（§八 策略二）

```
入场条件：周柱排 以"升"开头           → 周级方向为升
         位谕of周层下柱冲高提示 ≠ ""    → 有冲高信号
```

### 日诱甄别参考

| 日诱规则 | 柱排判断 |
|:--------|:---------|
| ② 主升后连阴回归DJA+大阳柱 | 近期柱排含"跌"链，然后突然转向"升" |
| ③ DJA之上W(连阴/跌连)/阴阳阴+鼎 | 柱排含"W"(连阴/跌连)模式，且柱型含"鼎" |
| ⑤ 反复围绕DJA震荡 | 柱排在"升"和"跌"之间频繁切换 |

## 七、柱排→数值信号编码表（回测框架直接引用）

### 7.1 匹配规则与优先级

柱排信号按下表优先级从高到低匹配。**首次匹配即命中**，不继续下沉。

M2.4 查表逻辑示例：
```python
def 柱排信号(柱排字符串):
    # 按表顺序匹配
    for rule in 信号编码表:
        if re.match(rule.模式, 柱排字符串):
            return rule.信号值
    return 0  # 未匹配时归零
```

### 7.3 信号否决规则（均线/趋势条件优先级高于柱排信号）

柱排信号值确定后，需经以下过滤得出**最终信号值**：

| 过滤条件 | 前置信号 | 否决后的最终信号 | 依据 |
|:---------|:--------:|:----------------:|:-----|
| `WXCD` = 屎/尿 | 任意值 | **0**（强制归零） | B §一(1) |
| `日类BTZE ≤ 0`（DJE下方） | 正值 | **0**（禁止做多） | B §二(7) |
| `日类BTZE ≤ 0`（DJE下方） | +1~+3 | 参考D表判断止跌阶段，**不做多** | B §二(7) |
| `日类BTZD ≤ 0`（DJD下方） | +3 | **+1**（降级：强势信号减弱） | B §三(一) |
| `WXAB < 0`（负交） | +2~+3 | **+1**（降级：仅可试仓） | B §二(5) |
| `WXZC ≤ 0` | 任意值 | **0**（禁止新开仓） | B §一(2) |

### 7.4 信号值语义

| 信号值 | 操作含义 | 对应仓位 |
|:------:|:---------|:---------|
| **+3** | 强势升链，可持有/加仓 | 基仓+浮仓 |
| **+2** | 买入信号或升链延续 | 基仓+浮仓（确认后） |
| **+1** | 升势可能减弱，持有关注 | 仅基仓 |
| **0** | 无方向/需确认/被否决 | 不操作 |
| **-1** | 跌势可能转强，关注反转 | 不买入，可观察 |
| **-2** | 卖出信号或跌链延续 | 卖出/不买入 |
| **-3** | 强势跌链，卖出/不买入 | 卖出/不持有 |

### 7.5 回测代码中引用此表的方式

```python
# M2.4 运行回测.py 引用示例
信号编码表 = [
    # (模式正则, 信号值, 名称)
    (r'升\.尾连[^ovVO]*$',      +3, '升链强势'),
    (r'升\.尾(连|吞)[ovVv]+Q',  +2, '升链含停顿'),
    (r'升\.尾吞',               +2, '升吞买入'),
    (r'升\.尾反孕[^O]*$',       +1, '升势减弱'),
    (r'跌\.尾连[^oOQv]*$',      -3, '跌链强势'),
    (r'跌\.尾(连|吞)[oOvV]+W',  -2, '跌链含停顿'),
    (r'跌\.尾吞',               -2, '跌吞卖出'),
    (r'跌\.尾反孕[^O]*$',       -1, '跌势减弱'),
    (r'\(升\)人\.连后吞.*Q$',   +2, '人排反转向上'),
    (r'\(升\)人\.连后吞',       0, '人排需确认'),
    (r'\(跌\)人\.连后吞.*W$',  -2, '人排反转向下'),
    (r'\(.*\)人\.',              0, '人排'),
    (r'错\.',                    0, '异常'),
]

def 过滤信号(柱排信号值, 日类BTZE, 日类BTZD, WXCD, WXAB, WXZC):
    """应用§7.3 信号否决规则"""
    if WXCD in ('屎', '尿'): return 0
    if WXZC <= 0: return 0
    if 日类BTZE <= 0: return max(柱排信号值, 0) if 柱排信号值 < 0 else 0
    # DJE上方但DJD下方，+3降级为+1
    if 日类BTZD <= 0 and 柱排信号值 == 3: return 1
    # WXAB负交时，+2~+3降级为+1
    if WXAB < 0 and 柱排信号值 >= 2: return 1
    return 柱排信号值
```
