## 一、四域判定逻辑

### 原子条件

| 概念 | 条件 | VBA变量 | 算展列 |
|:----|:-----|:--------|:-------|
| WXZC>0 | 周类BTZC > 0 | `周类BTZC` | BTZC 周 |
| DXZE>0 | 日类BTZE > 0 | `日类BTZE` | BTZE 日 |
| WXCD>0（金） | 周类BTZC>0 且 周类BTCD>0 | `周类BTZC > 0 And 周类BTCD > 0` | — |
| WXZB>0（甲乙己） | 周类BTZB > 0 | `周类BTZB > 0` | BTZB 周 |
| JE上JF | JE > JF | `日类JE > 日类JF` | — |

### 四域周

```
IF 周类BTZC > 0 THEN            '满足门槛
    IF BTZC>0 AND BTCD>0 AND BTZB>0 THEN   '金 + 甲乙己
        结果 = "多长"
    ELSE
        结果 = "多被"
    END IF
ELSE
    IF BTZC<=0 AND BTCD<=0 AND BTZB<=0 THEN  '全部非正
        结果 = "空长"
    ELSE
        结果 = "空看"
    END IF
END IF
```

### 四域日

```
IF 日类BTZE > 0 THEN            '满足门槛
    IF BTZE>0 AND JE>JF AND BTZD>0 THEN    '金 + 甲乙己（日版）
        结果 = "多长"
    ELSE
        结果 = "多被"
    END IF
ELSE
    IF BTZE<=0 AND JE<=JF AND BTZD<=0 THEN  '全部非正
        结果 = "空长"
    ELSE
        结果 = "空看"
    END IF
END IF
```

## 二、日层段（DJEDC之上仓位状态）

```
IF 日类BTZE <= 0 THEN    'DJE之下，非操作区域
    结果 = ""
ELSEIF 日类BTZC <= 0 THEN  'DJE之上但DJC之下
    结果 = "卖浮"          '浮仓退出
ELSEIF 日类BTZA > 0 THEN   'DJE之上且DJC之上且DJA之上
    结果 = "持主"          '主动持有
ELSE                        'DJE之上且DJC之上但DJA之下
    结果 = "持被"          '被动持有
END IF
```

## 三、日机警（概率提示）

### 警止盈

```
IF 区划=_多长 AND DJA之上 AND 管宽周偏哼JA>=20 THEN
    结果 = "警止盈"
```

### 警诱多

```
IF 区划=_多长 AND DJA之上 AND (
    上破DJA三柱内 OR 连阴后阳 OR 阴阳阴后阳
) THEN
    结果 = "警诱多"
```

其中：
- 上破DJA三柱内 = 日类BTZA>0 AND 前一日日类BTZA≤0
- 连阴后阳 = 前一日柱排为跌连 AND 今日涨幅>0
- 阴阳阴后阳 = 前两日为阴阳阴模式 AND 今日涨幅>0

### 漏日

```
IF 区划=_多长 AND DJA之上 AND 日柱排条件 THEN
    结果 = "漏日"
```

### 漏周

```
IF 区划=_多长 AND DJA之下 AND 周柱排为升开头 THEN
    结果 = "漏周"
```

## 四、仓位退出规则

```
基仓退出：WXZC < 0（等价于DJE跌破）
    → 全部清仓，无一例外

浮仓退出：下破DJC（日类BTZC <= 0）
    → 仅平浮仓，保留基仓

周浮仓时间限制：周五收盘前无论盈亏平仓
日浮仓时间限制：次日收盘前无论盈亏平仓
```
