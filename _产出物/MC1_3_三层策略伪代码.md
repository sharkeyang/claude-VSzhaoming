## 一、核心条件速查

| 概念 | VBA变量全称 | 含义 |
|:-----|:-----------|:-----|
| **WXCD** | `周类BTCD` | W周级 + CD(JZ与JD的交叉天数) |
| **WXZC** | `周类BTZC` | W周级 + ZC(JZ与JC的交叉天数) |
| **WXAB** | `周类BTAB` | W周级 + AB(JA与JB的交叉天数) |
| **WXZA** | `周类BTZA` | W周级 + ZA(JZ与JA的交叉天数) |
| **DXZE** | `日类BTZE` | D日级 + ZE(JZ与JE的交叉天数) |
| **DXZD** | `日类BTZD` | D日级 + ZD(JZ与JD的交叉天数) |
| **DXZC** | `日类BTZC` | D日级 + ZC(JZ与JC的交叉天数) |
| **DXZB** | `日类BTZB` | D日级 + ZB(JZ与JB的交叉天数) |
| **DTZA（已站稳）** | `日类BTZA` | ≥3 即在DJA之上站稳 |
| **DXCD** | `日类BTCD` | D日级 + CD(JZ与JD的交叉天数) |
| **DXAB** | `日类BTAB` | D日级 + AB(JA与JB的交叉天数) |
| **DSHA** | `位谕of日层BSHA` | (今高/DJA - 1) × 100，≥6即偏 |
| **柱排周** | `位谕of周层柱排` | 周级柱排字符串 |
| **柱排日** | `位谕of日层柱排` | 日级柱排字符串 |
| **柱型日** | `位谕of日层柱型` | 日级柱型字符串 |
| **盈提示** | `位谕of日层盈提示` | 止盈信号串（雀/宽/高/偏/连） |
| **周层盈丘幅** | `位谕of周层盈丘幅` | (龟结顶-WJC)/WJC × 100 |

## 二、前置过滤器（准入条件）

> 任意标的在进入三层策略前，必须通过此过滤器。某一条未通过即淘汰。

```pseudo
FUNCTION 通过前置过滤(标的):
    ' ===== 硬条件①：WXCD 状态 =====
    IF 周类BTCD <= 0 THEN RETURN 淘汰("WXCD非金非银")
    
    ' ===== 硬条件②：周线分界 =====
    IF 周类BTZC <= 0 THEN RETURN 淘汰("前周WXZC<=0")
    
    ' ===== 硬条件③：日线准入 =====
    IF 日类BTZE <= 0 THEN RETURN 淘汰("今日DXZE<=0")
    
    ' ===== 硬条件④：日条件 =====
    ' DXCD最好甲乙己，可接受其他。本质是DXZD>0
    IF 日类BTZD <= 0 THEN RETURN 淘汰("DXZD<=0")
    
    RETURN 准入
END FUNCTION
```

## 三、三层策略引擎

### 框架概览

```
对每个通过前置过滤的标的：

  策略一（基仓长期持有）
    ├── 乾坤条件满足 → 纳入基仓候选
    ├── DJED 日主战场（DJE之上即日主战场，坐公交容忍日线波动）
    ├── 分清级别：日线波动看周线定夺
    ├── 止盈：位谕of日层盈提示含高/偏/宽→部分止盈
    ├── 止损①：日类BTZE<0 → 全部清仓（最高优先级）
    └── 止损②：确认过最高点后周类BTZB<0 → 基仓清仓

  策略二（下周冲高）——周五触发
    ├── 前提：基仓已持有
    ├── 周柱排以"升"开头
    ├── 位谕of周层下柱冲高提示 NOT ""
    └── 尾盘入场，下周五无论盈亏平仓

  策略三（下日冲高）——周中触发
    ├── 前提：基仓已持有
    ├── 日柱排以"升"开头
    ├── 日类BTZA > 0（收盘价在DJA之上）
    ├── 诱多甄别5条全部通过
    └── 尾盘入场，次日不及预期即退
```

### 策略一：基仓长期持有（乾坤条件）

```pseudo
FUNCTION 基仓策略(标的):
    ' ===== 入场条件 =====
    ' 乾坤条件：BTZD>0 AND BTZC>0 AND BTZB>0
    '（对应文档概念：DXZD>0 AND DXZC>0 AND DXZB>0）
    IF 日类BTZD <= 0 THEN RETURN 不入基仓("乾坤ZD不满足")
    IF 日类BTZC <= 0 THEN RETURN 不入基仓("乾坤ZC不满足")
    IF 日类BTZB <= 0 THEN RETURN 不入基仓("乾坤ZB不满足")
    
    ' 仓位：基仓 5000 固定
    基仓量 = 5000
    
    ' ===== 持有监控（每交易日执行） =====
    ' DJE之上 = 日主战场（生命线之上，趋势完好）
    ' DJED 安心持有：DJE与DJD之上即日主战场
    ' DJC/DJB以下的波动属"坐公交"范畴，不构成退出理由
    ' 分清级别：日线波动看周线定夺，只有周类BTZA<0才需要关注
    
    ' 🔴 日线视角空头排列检查
    ' DJE之下(V0忑~V9下)：仓日类统一"无", 做空与做空止损
    ' DJE之上(E0上~E9忐)：仓日类{护级CD}{护段AB}, 按等级判断牢固度
    ' 日周联动列位2现直接显示 V/E 等级码（如E0上/V3忑），取代原仓日类值
    ' 上破 DJA/DJB 不等于买入信号——只是止跌
    ' 空头止损：上破DJE(严格)→上破DJD(放宽)→起码上破DJC
    IF 日类BTZE <= 0 THEN
        ' DJE之下，空头格局
        ' 不做多，只做空与做空止损
        RETURN 基仓不持有("DJE之下，空头格局，不满足基仓条件")
    END IF
    
    IF 日类BTZE > 0 AND 日类BTZD > 0 THEN
        ' 🔵 DJED 日主战场 DJEDCB安心持有，分清级别
        ' 无视日级别 DJC/DJB 以下的波动
        ' 只有当周级别也走弱时才关注：
        IF 周类BTZA < 0 THEN
            ' 周线视角也走弱，标记关注但不操作
            RETURN 基仓持有(状态:"关注", 动作:"不动")
        ELSE
            RETURN 基仓持有(状态:"安心", 动作:"不动")
        END IF
    END IF
    
    ' ===== 止盈（非卖出，是部分了结） =====
    ' 参考位谕of日层盈提示：
    ' "高" → 高幅≥3应止盈
    ' "偏" → BSHA≥6，偏离过大需止盈
    ' "宽" → 哼JC宽度过大
    ' "雀"/"连N" → 连阳信号
    IF 盈提示 包含 "高" OR 盈提示 包含 "偏" OR 盈提示 包含 "宽" THEN
        RETURN 基仓建议(动作:"止盈部分", 说明:"盈提示触发")
    END IF
    
    ' 镜面对称：不要因为DJA之上一根小阴柱就卖出
    ' 只要日柱排不是"跌"开头，且不是连阴/阴阳阴模式，阴柱忽略
    IF 日柱排 以"跌"开头 THEN
        ' 确认为跌排，需进一步判断
        ' 但即使跌排，只要未破DJE，基仓不动
        ' 基仓不动，仅影响浮仓判断（见策略三）
    END IF
    
    ' ===== 止损（刚性执行） =====
    ' ① DJE跌破 → 最高优先级，全部清仓
    IF 日类BTZE < 0 THEN
        RETURN 基仓清仓("DJE跌破，全部清仓，无一例外")
    END IF
    
    ' ② WXZC<0 → 终极止损
    ' 周级别方向走弱（WXZC<0），趋势终结，基仓止损清仓
    IF 周类BTZC < 0 THEN
        IF 已经确认经过最高点 THEN
            RETURN 基仓清仓("WXZC<0，趋势终结，基仓止损")
        END IF
    END IF
    
    RETURN 基仓持有(状态:"正常", 动作:"不动")
END FUNCTION
```

### 策略二：下周冲高策略（周五尾盘触发→为下周布局）

```pseudo
FUNCTION 下周冲高策略(标的):
    ' ===== 前置条件 =====
    ' 基仓必须已持有
    IF 未持有基仓(标的) THEN RETURN 不触发("无基仓")
    
    ' ===== 触发时机：周五尾盘 =====
    IF 今日不是周五 THEN RETURN 不触发("非周五")
    IF 当前时间 < 14:30 THEN RETURN 等待尾盘
    
    ' ===== 入场条件 =====
    ' ① 周柱排以"升"开头
    IF 周柱排 不以"升"开头 THEN RETURN 不触发("周柱排非升开头")
    
    ' ② 下柱冲高提示不为空
    IF 位谕of周层下柱冲高提示 = "" THEN RETURN 不触发("无下柱冲高提示")
    
    ' ===== 仓位 =====
    浮仓量 = 2000  ' 下周冲高仓（上限2000）
    ' 禁忌检查：浮仓总量不能超过基仓
    IF 当前总浮仓 + 2000 > 基仓量 THEN RETURN 不触发("浮仓超限")
    
    ' ===== 入场执行 =====
    执行买入(标的, 量:=浮仓量, 类型:="下周冲高")
    
    ' ===== 出场条件（下周监控） =====
    ' 🟢 止盈：下周涨幅达标 → 平仓
    ' 🔴 止损：下破 DJB（日类BTZB<0）→ 平仓
    ' ⏰ 时间限制：如果下周未触发止盈/止损，周五无论盈亏平仓
END FUNCTION

FUNCTION 下周冲高出场监控(标的):
    ' 每日检查
    IF 日类BTZB < 0 THEN
        执行卖出(标的, 原因:"DJB止损")
        RETURN
    END IF
    
    ' 临涨幅达标 → 止盈
    IF 下周累计涨幅 >= 目标止盈 THEN
        执行卖出(标的, 原因:"下周冲高止盈")
        RETURN
    END IF
    
    ' 周五无论盈亏平仓
    IF 今日是周五 THEN
        执行卖出(标的, 原因:"下周冲高到期平仓")
        RETURN
    END IF
END FUNCTION
```

### 策略三：下日冲高策略（周中尾盘触发→需剔除日诱）

```pseudo
FUNCTION 下日冲高策略(标的):
    ' ===== 前置条件 =====
    ' 基仓必须已持有
    IF 未持有基仓(标的) THEN RETURN 不触发("无基仓")
    
    ' ===== 触发时机：周一到周四尾盘 =====
    IF 今日是周五 THEN RETURN 不触发("周五走下周冲高策略")
    IF 当前时间 < 14:30 THEN RETURN 等待尾盘
    
    ' ===== 前置检查：甄别已冲高 =====
    ' DSHA 追高限制：BSHA ≥ 6 代表已冲高，只持基仓不加浮仓
    IF 位谕of日层BSHA >= 6 THEN
        RETURN 不触发("DSHA偏高，已冲高标的只持基仓")
    END IF
    
    ' ===== 入场条件 =====
    ' ① 日柱排以"升"开头
    IF 日柱排 不以"升"开头 THEN RETURN 不触发("日柱排非升开头")
    
    ' ② 收盘价在 DJA 之上
    IF 日类BTZA <= 0 THEN RETURN 不触发("收盘价未在DJA之上")
    
    ' ③ DTZA≥3 已站稳DJA（非刚突破）
    ' DTZA = 日类BTZA，≥3代表在DJA之上连续3天以上，已确认站稳
    ' <3代表刚刚站上或还在震荡，不开仓
    IF 日类BTZA < 3 THEN RETURN 不触发("DTZA<3，DJA尚未站稳")
    
    ' ④ 通过诱多甄别（5条全过）
    IF NOT 通过日诱甄别(标的) THEN RETURN 不触发("命中日诱规则")
    
    ' ===== 仓位 =====
    浮仓量 = 2000  ' 下日冲高仓（上限2000）
    IF 当前总浮仓 + 2000 > 基仓量 THEN RETURN 不触发("浮仓超限")
    
    ' ===== 入场执行 =====
    执行买入(标的, 量:=浮仓量, 类型:="下日冲高")
    
    ' ===== 出场条件 =====
    ' 🟢 止盈：次日冲高3%平仓
    ' 🔴 止损：下破 DJD（日类BTZD<0）平仓
END FUNCTION

FUNCTION 下日冲高出场监控(标的):
    ' 次日检查
    IF 日累计涨幅 >= 3% THEN
        执行卖出(标的, 原因:"下日冲高止盈达标")
        RETURN
    END IF
    
    IF 日类BTZD < 0 THEN
        执行卖出(标的, 原因:"下破DJD止损")
        RETURN
    END IF
    
    ' 如果次日未达标也未止损，继续持有
    ' 但持续监控，直到触发止盈/止损
    ' 且不能转为"长期持有"——冲高仓做完即退
END FUNCTION
```

## 四、日诱甄别（5条剔除规则）

```pseudo
FUNCTION 通过日诱甄别(标的) -> 布尔:

    ' ===== 规则①：DXAB=<己>负交+上破DJA+DJB =====
    ' DXAB=<己>：日类BTAB≈0（刚刚交叉或在交叉边缘）
    ' DXAB负交：日类BTAB<0（实际上条中己就是负交或即将正交）
    ' 同时上破DJA与DJB：JZ同时站上JA和JB
    IF 日类BTAB <= 0 AND 日类BTZA > 0 AND 日类BTZB > 0 THEN
        ' 看似突破，实为诱多
        RETURN 不通过("规则①：DXAB<己>负交同时上破DJA与DJB")
    END IF
    
    ' ===== 规则②：主升后连阴回归DJA+大阳柱 =====
    ' 主升后以连阴方式跌回DJA附近
    ' 随后出现大阳柱 → 反弹诱多，非趋势延续
    ' 判断依据：日柱排历史序列
    ' 近期柱排含"连阴"或"阴阳阴"模式，且最新柱排为阳/升
    IF 日管中符串 含连阴模式 AND 日柱排 以"升"开头 THEN
        RETURN 不通过("规则②：主升后连阴回归DJA+大阳柱")
    END IF
    
    ' ===== 规则③：DJA之上连阴/阴阳阴+鼎 =====
    ' 在DJA之上以连阴或阴阳阴形态出现
    ' 随后从DJA之下上穿DJA出现"鼎"
    IF 位谕of日层BT鼎 = 1 THEN
        IF 日管中符串 含连阴模式 OR 日管中符串 含"阴阳阴" THEN
            RETURN 不通过("规则③：DJA之上连阴/阴阳阴+鼎")
        END IF
    END IF
    
    ' ===== 规则④：下破DJA须等再站上 =====
    ' 不能以下破DJA作为入场信号
    ' 要等价格重新站上DJA才算信号
    ' 即：日类BTZA 从负变正的当天或之后
    IF 日类BTZA <= 0 THEN
        ' 当前在DJA之下，即使柱排=升开头也不开仓
        RETURN 不通过("规则④：当前在DJA之下，须等再站上")
    END IF
    
    ' ===== 规则⑤：反复围绕DJA震荡 =====
    ' DTZA≥3=已站稳DJA，不需要这条
    ' 但如果1≤日类BTZA<3，说明刚站上不久
    ' 结合柱排看是否反复围绕DJA
    IF 日类BTZA < 3 THEN
        IF 日管中符串 含围绕DJA震荡模式 THEN
            RETURN 不通过("规则⑤：DJA附近反复震荡")
        END IF
    END IF
    
    ' 全部通过
    RETURN 通过
END FUNCTION
```

## 五、浮仓管理规则

```pseudo
FUNCTION 浮仓管理(标的):
    ' ===== 浮仓开仓条件 =====
    ' 基仓已持有 + DJA上方阴柱
    IF 未持有基仓(标的) THEN RETURN 不可开浮仓("无基仓")
    IF 日类BTZA <= 0 THEN RETURN 不可开浮仓("DJA之下不开浮仓")
    
    ' DJA上方收出阴柱才可开浮仓
    ' "阴柱" = 柱排含"跌"或"阴"字面，但需结合坐公交镜面对称：
    ' 镜面对称❸：DJA之上阴柱不代表转跌
    ' 只要不是"跌"开头的柱排，且不是连阴/阴阳阴，阴柱可忽略
    ' 但浮仓开仓需要阴柱信号本身（阴柱=买入机会）
    IF NOT (柱排日 含阴柱信号) THEN
        RETURN 不可开浮仓("无DJA上方阴柱信号")
    END IF
    
    ' ===== DSHA 追高限制 =====
    IF 位谕of日层BSHA >= 6 THEN
        RETURN 不可开浮仓("DSHA偏高，只持基仓")
    END IF
    
    ' ===== 浮仓平仓条件 =====
    ' ① DXZC<0 → 平所有浮仓（日类BTZC<0）
    IF 日类BTZC < 0 THEN
        执行平浮仓(标的, 原因:"DXZC<0，浮仓全平")
        RETURN
    END IF
    
    ' ② 行情回归平稳 → 缩减回基仓
    ' （由持仓决策树判断）
    
    ' ③ 下破DJC → 平浮仓（日类BTZC<0 且 日类BTCD<0）
    IF 日类BTZC < 0 AND 日类BTCD < 0 THEN
        执行平浮仓(标的, 原因:"跌破DJC+DXCD<0，只平浮仓保留底仓")
        RETURN
    END IF
END FUNCTION
```

## 六、浮仓禁忌（日诱同步）

```pseudo
FUNCTION 浮仓禁忌检查(标的) -> 布尔:
    ' 下列任一情况 → 禁止加浮仓
    
    ' 1. 连阴时不要加仓
    IF 日管中符串 含连阴模式 THEN RETURN 违反禁忌
    
    ' 2. 连阴后大阳不要追
    IF 前日柱排 含连阴模式 AND 今日柱排 含大阳模式 THEN RETURN 违反禁忌
    
    ' 3. 阴后小鼎不要碰
    IF 位谕of日层BT鼎 = 1 AND 前日柱排 含阴模式 THEN RETURN 违反禁忌
    
    ' 4. 不要下破DJA又上破DJA（DTZA计数）
    IF 日类BTZA < 3 THEN
        ' 刚站上DJA不久，容易被震出去
        IF 近N日有围绕DJA的上下波动 THEN RETURN 违反禁忌
    END IF
    
    ' 5. 围绕DJA反复震荡 → 直接放弃
    ' 同规则④⑤的日诱甄别
    
    RETURN 无禁忌
END FUNCTION
```

## 七、坐公交理论 + 镜面对称（决策流中的位置）

### 坐公交理论 — 插入基仓决策树

```
位置：策略一 基仓持有监控 中
条件：
  DJED 日主战场（日类BTZE>0 AND 日类BTZD>0）
    → DJC/DJB以下的波动忽略
    → 仅当周类BTZA<0时才关注日线
  DJE跌破 → 全部清仓（这是止损，不是"被震出去"）
```

### 镜面对称❶ 小跌不卖 — 插入基仓决策树

```
位置：策略一 持有监控
条件：
  日类BTZE > 0 → 未破DJE，算"小跌"，基仓不动
  不因日级别的小幅下跌而退出基仓
```

### 镜面对称❷ 不到WJB不卖 — 插入基仓止损

```
位置：策略一 止损判断（最后一道防线）
说明：
  这不是止盈，是趋势止损。
  确认经过最高点后，下破WJB（周类BTZB<0）= 趋势终结
  在此之前，不因主观判断"这票到顶了"而卖出
  "舍不得止盈"是对的——没破WJB就不卖
```

### 镜面对称❸ DJA阴柱不代表转跌 — 插入基仓+浮仓决策树

```
位置：基仓持有监控 + 浮仓判断
条件：
  日柱排不是以"跌"开头 → 阴柱忽略
  不是"连阴"或"阴阳阴"模式 → 阴柱忽略
  DJA之上出现单根小阴柱，继续持有不动
  适用于基仓和浮仓
```

## 八、执行流程时间线

```pseudo
' ===== 交易日流程 =====

FUNCTION 每日操盘流程(标的池):
    ' 14:00 前：数据更新
    更新所有指标(标的池)
    
    ' 14:00 - 14:30：AI决策
    FOR EACH 标的 IN 标的池
        IF NOT 通过前置过滤(标的) THEN 跳过(标的)
        
        ' ① 基仓策略（对所有持仓）
        基仓决策 = 基仓策略(标的)
        
        ' ② 浮仓管理
        浮仓管理(标的)
        
        ' ③ 周中 → 下日冲高判断
        IF 今日非周五 THEN
            下日冲高策略(标的)
        END IF
        
        ' ④ 周五 → 下周冲高判断
        IF 今日是周五 THEN
            下周冲高策略(标的)
        END IF
        
        ' ⑤ 下周冲高出场监控（对已有的下周冲高持仓）
        下周冲高出场监控(标的)
        
        ' ⑥ 下日冲高出场监控（对已有的下日冲高持仓）
        下日冲高出场监控(标的)
    NEXT
    
    ' 14:30 后：输出操盘计划 → 人/自动化执行
    输出操盘计划()
END FUNCTION
```

| 下日冲高止盈 | 次日涨幅 ≥ 3% | 达标即平 |
| 下周冲高持仓 | 最长到周五 | 无论盈亏，周五平仓 |
