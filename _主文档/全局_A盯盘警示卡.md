## 二、持仓生死线（刚性执行）

### 2.1 持仓核心规则

| |
|:--|
| 🔴 **`DJE` 是唯一止损线** 价格有效跌破 `DJE` → 全部仓位无条件清仓<br>（无论 `DXCD`/`DXAB` 正负，无论任何指标）<br>*口诀：不破 `DJE` 不离场，破线无条件清盘* |
| 🟡 **`DJC` 只做入场参考，绝不用来止损** 跌破 `DJC` ≠ 卖出信号<br>但：`DXCD < 0` AND 跌破 `DJC` = 只平浮仓，保留底仓 |
| 🟢 **不破 `WJB` 不退**，可以保证经过最高点 |
| 🟠 **提前止盈（唯一情况）** 只有行情大幅冲高之后，才可主动止盈<br>→ 无冲高不提前卖出 |
| 🟢 **常态持仓** `DJE` 之上即 **日主战场**。`DJEDCB` 区间安心持有，分清级别：日线波动看周线定夺。不提前止盈 |

📎 完整版：`全局_B交易体系.md` §四、§六(2)

### 2.2 为什么日线必须用 `DJE` & `DJD` 过滤（核心原理）

> **盯盘时冲动想下单，先读这节。理解原理才能守住纪律。**

| |
|:--|
| 🟢 **基本原因：** `DJE`（= `WJC` 映射）之上 = 可以做多 ✅<br>`DJD` 之上 = 保证 `DXCD` 为甲乙己 ✅<br>→ `DJE` & `DJD` 是日线做多的基本条件，缺一不可 |
| 🟡 **形态原因：** 日线主跌之后 → 日震荡 → 从 `DJA` 上破 → `DJD` → `DJE`<br>这个过程中 `DJFEDCBA` 负交 → 很可能再次形成空头排列 ❌<br>→ 应当规避这些震荡过程，等待站上 `DJE`&`DJD` 再操作 |
| 🔴 **大势原因（阴阳对称）：** 主升之后 = 震荡出货（不参与）<br>主跌之后 = 震仓吸货（不参与）<br>只有主升 = 值得操作 ✅ |
| 🔴 **结论：** `DXEF` 负交 → `DJE`&`DJD` 之下 ❌ 绝对禁止参与<br>`DXEF` 正交 → `DJE`&`DJD` 之上 ✅ 坚守主战场<br>中间过渡区 → 以 `DJE`&`DJD` 为准，不要幻想 |

> **理解了这个，你就不会在 `DJE` 之下抄底了。你以为抄的是底，其实是震荡出货的上影线。**

### 3.5 防二戒（保持基仓的SOP）

> **单向二戒：在同一个位置只能做一个方向，盈利退出后禁止立刻反向操作。**

| 戒律 | 操作指南 |
|:----|:---------|
| 🚫 **一戒** | 盈利退出后，不因偏离过大或参考低阶波动而立刻反向操作。**盈利不是反向的理由。** |
| 🚫 **二戒** | 趋势判断正确后，不要过早退出。趋势没有结束，**底仓（基仓）不要动。** |
| 🚫 **止损退出** | 因 `DJE` 止损清仓后，不准在同日或次日反向开仓。至少等待重新站上 `DJE`。 |
