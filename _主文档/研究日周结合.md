## 一、核心原则

> **先周后日。** 日服从周，周级别决定方向，日级别负责择时。

| 层级 | 职责 | 决策依据 |
|:----|:-----|:---------|
| 🔴 **周级别** | 定方向、定可否操作 | `WXZC>0` + `WXAB>0` |
| 🟢 **日级别** | 定时机、定点位 | `DXZC>0` + `DXAB>0`（在周级别框架内） |

**不存在脱离周级别的日级别独立操作。**

## 二、镜面对称：上升过程 vs 下跌过程

整个系统的核心洞见是：**上升过程的规则与下跌过程的规则互为镜面对称**。

### 2.1 上升过程（DJCBA多头排列）

```
条件：DJC > DJB > DJA（多头排列），DXZC > 0 且 DXAB > 0
```

| 位置 | 规则 | 错误倾向 |
|:----|:-----|:---------|
| DJA之上，阳柱 | 不要轻易止盈 | 一涨就想卖 |
| DJA之上，阴柱 | **禁止止损** | 一跌就想跑 |
| 下破DJA | **不构成止损，不要随意退出** | 把DJA当止损线 |
| 下破DJB | **不构成止损，不要随意退出** | 把DJB当止损线 |
| 满足日类退出条件（`DXZC<0 且 DXAB<0`） | 可暂时退出，也可不退出 | 应该结合周级别判断 |

> **核心信念：** 在上升趋势中，DJA之上的阴柱根本不用怕。连下破DJB都不构成退出理由。

### 2.2 下跌过程（DJCBA空头排列）

```
条件：DJC < DJB < DJA（空头排列），DXZC < 0 且 DXAB < 0
```

| 位置 | 规则 | 错误倾向 |
|:----|:-----|:---------|
| DJA之下 | **不要介入** | 抄底思维 |
| 仅上破DJA | **不要介入** | 以为反转 |
| 与基本介入条件还有距离 | **不要介入** | 抢跑 |
| 周级别未满足（`WXZC<0 且 WXAB<0`） | **必须等周级别满足后才可介入** | 日级别信号不足以独立决策 |

> **核心信念：** 在下跌趋势中，DJA之下的阳柱根本不要碰。连上破DJA都不能算信号。

## 三、先周后日：具体流程

### 3.1 日级别介入的前提

```
周级别条件（必须先满足）
├── WXZC > 0  （前周分界为正）
├── WXAB > 0  （周级持仓为正交）
└── WXCD ≠ 屎尿（大势不差）
        ↓
日级别条件（在周级别框架内）
├── DXZC > 0  （日级DJC之上）
├── DXAB > 0  （日级AB正交）
├── DXZA > 0  （日级DJA之上）
└── DXZE > 0  （日级DJE准入）
```

### 3.2 上升趋势中的日级别操作

当周级别进入上升过程后，开始考虑日级别 `DXAB`：

| 场景 | 规则 |
|:----|:-----|
| DJCBA多头排列，在DJA之上 | 日柱排列逐渐变化，可能出现启动 |
| 周主升快速拉升阶段 | 以日不破DJA/DJB为辅助判断依据 |
| DJCBA多头排列，DXZA动，在DJA之下 | 日柱可能不按均线束缚，无规律 |
| 处于DXAB护型乙，待选择方向 | 如果在DJA之下介入，必须有周柱组合配合 |

### 3.3 下跌趋势中的日级别退出

```
WXZC>0且WXAB>0 → DXZC<0且DXAB<0
     ↑                    ↑
   周条件仍在           日条件已变
     ↓                    ↓
   可在此退出的场景：
   ├── 周下破WJA → 可退出
   ├── 若之前未冲高 → 可以DJD作退出线
   ├── 下破DJC且DXAB<0 → 最佳退出点（下跌过程开始）
   └── 下破DJD → 必须退出
```

### 3.4 调整后的再介入

经调整后再次满足 `DXZC>0 且 DXAB>0`：
- 须参考周类
- 若认为有冲高可能，可再介入
- 若DXAB未正交（`DXAB<0 且 DXZB>0`），不一定要介入，仅可试仓
- 应等待DXAB正交后再介入，且应保证DXAB持续较长时间

## 四、日级别操作全景对照

### 4.1 从周级别向下看

```
WXZC>0 且 WXAB>0（周级上升）
  │
  ├── DJCBA多头排列，在DJA之上 ← 最安全，正常操作
  │     ├── 阳柱 → 持有，不轻易止盈
  │     └── 阴柱 → 不构成止损，不退
  │
  ├── DJCBA多头排列，在DJA附近
  │     ├── 突破回踩 → 机会
  │     └── 窄管蓄力 → 机会
  │
  ├── DJCBA多头排列，短暂下破DJA
  │     └── 不构成退出理由，继续持有
  │
  └── DXZC<0 且 DXAB<0（日条件恶化）
        ├── 可暂时退出
        ├── 也可不退（看周级别）
        └── 周下破WJA → 必须退
```

### 4.2 从日级别向上看

```
日级别信号出现
  │
  ├── 检查周级别：WXZC>0？WXAB>0？WXCD≠屎尿？
  │     ├── 是 → 日级别信号可执行
  │     └── 否 → 日级别信号直接放弃
  │
  └── 检查周级别走势阶段
        ├── 周主升快速拉升 → 以日不破DJA/DJB为辅助依据
        ├── 周震荡上升 → 日级别正常操作
        └── 周级别转弱 → 日级别只减不加
```

## 六、结论

**镜面对称框架是对已有规则的结构化总结，没有冲突。** 它最大的价值在于：

1. **提供心理锚点** — "上升中阴柱别怕，下跌中阳柱别碰"这个镜面对称思维可以扛住情绪波动
2. **强化先周后日** — 任何日级别决策都必须先过周级别过滤
3. **明确退出层级** — DJA/DJB/DJC不是退出线，DJE和WJA才是

> **一句话：** 周级给方向，日级给时机。上升中DJA之上阴柱不构成退出理由，下跌中DJA之下阳柱不构成介入理由。两头对称，中间自由。

## 七、日级别介入决策清单

> **来源：** `图片日级别是否应介入.png` — 日级别介入时机的系统性决策框架
> **前置条件：** 以下所有决策均需先过周级别过滤（`WXZC>0 且 WXAB>0`），周级不满足则直接放弃。

### 7.1 ✅ 应介入

| # | 场景 | 关键条件 | 说明 |
|:---:|:----|:---------|:-----|
| 1 | **初始突破DJA之上阴柱** | 刚突破DJA不久 | 阴柱通常是上升中继，不意味下破DJA |
| 2 | **上破回踩** | DXZA/DXZB后，以跌孕/跌连回踩DJA；DTZA=2/3 | 形成启动结构的标志 |
| 3 | **贴附DJA运行** | 管宽较小，沿DJA行进 | 最稳定情况，如行进队列 |
| 4 | **连升后阴柱** | 阴柱回归DJA或下破DJA | 可能立刻反包，是机会 |
| 5 | **连升后跌孕/跌吞/下破DJA** | 特别是合顶情况 | 很可能是机会 |
| 6 | **跌连后** | 前期未冲高，下跌角度小，十字星平移下跌 | 急跌后缓跌，见底信号 |
| 7 | **下破DJA仅为乙级护型** | 未到丙级、未达DXAB负交、未下破DJC | 遇支撑或转升阳柱可再介入 |
| 8 | **劫后转好** | DXZB/DXAB/DXAC交后，跌排失效，初始突破DJA | 调整结束信号 |
| 9 | **DXZA暂下破恢复** | 单柱/两柱下破DJA，或a-b-c浪调整 | 需确认恢复方向 |
| 10 | **充分调整后站上DJA** | a-b-c浪完成，再次站上DJA | 防止M头 |

### 7.2 ❌ 不应介入

| # | 场景 | 关键指标 | 风险 |
|:---:|:----|:---------|:----|
| 1 | **首上破DJA** | DSHA较大 或 首上破DJA | 禁入 — 无历史支撑，假突破风险高 |
| 2 | **已冲高** | DSHA较大 / BRDSHCEEA | 不应追高 |
| 3 | **高位震荡** | DSAC较大（已拉升过） | 上破DJA可能形成M头诱多 |
| 4 | **跌吞+升吞交替** | 顶部剧烈震荡 | 不在大阳后介入 |
| 5 | **右侧上破DJA** | 整体向下大方向未改 | 诱多——小反弹不改下降趋势 |
| 6 | **顶高横盘后上破DJA** | 横盘一段时间后 | 很可能诱多 |
| 7 | **连跌后出现继易** | 与前面连跌形成跌排 | 是下跌中继 |
| 8 | **下破DJA后再形成首刚** | 与之前下跌形成[人排] | 需等待站稳 |

### 7.3 ⚠️ 不确定（需结合上下文）

| # | 情境 | 矛盾点 | 判断方法 |
|:---:|:----|:-------|:---------|
| 1 | 下破DJA后开启新DJA丘 | 可能在下破阴柱后立刻开启新丘 | 观察阴柱后的第一根阳柱是否站上DJA |
| 2 | 下破DJA后升孕阳柱 | 可能是确认下破，也可能后续连升上破 | 看后续2~3柱是否站上DJA |
| 3 | 远高DJA逐步向DJA靠近 | 可能得到支撑反弹，也可能下破 | 看接触DJA时的柱形态 |
| 4 | 平移调整后触及DJA | 得支撑则继续上升，偏高较大则下破 | 看调整幅度和管宽 |
| 5 | 快速回归DJA后大阳反弹 | 可能偏离DSHA较大后下破 | 看反弹后的第二柱确认 |

### 7.4 决策流程

```
日级别信号出现
  │
  ├── 先过周级别过滤
  │     ├── WXZC>0? WXAB>0? WXCD≠屎尿?
  │     └── 不满足 → ❌ 放弃
  │
  ├── 再判断日级别位置
  │     ├── DJCBA多头排列，在DJA之上
  │     │     └── 正常操作（§7.1 #1~#5）
  │     ├── DJCBA多头排列，在DJA附近/短暂下破
  │     │     └── 看护型等级（§7.1 #6~#8）
  │     ├── 已冲高/高位震荡
  │     │     └── ❌ 禁入（§7.2 #1~#4）
  │     ├── 右側上破或横盘后上破
  │     │     └── ⚠️ 诱多警惕（§7.2 #5~#6）
  │     └── 下破DJA后的复杂情况
  │           └── ⚠️ 不确定（§7.3）
  │
  └── 检查周级别走势阶段
        ├── 周主升快速拉升 → 侧重§7.1 #3 贴附DJA
        ├── 周震荡上升 → §7.1 #1~#5均可
        └── 周级别转弱 → 只减不加
