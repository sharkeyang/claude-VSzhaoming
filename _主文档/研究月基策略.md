## 月基评分原则

月基评分用于评估一只股票是否适合长期持有（月基策略），分数越高越适合。

**评分公式：**



| 维度 | 说明 | 意义 |
|:----|:-----|:-----|
| **年化收益** | 多长→WXZC<0退出期间的年化收益率 | 策略整体盈利能力 |
| **胜率** | 盈利交易次数 / 总交易次数 | 策略的稳定性 |
| **盈亏比** | 平均盈利 / 平均亏损 | 策略的风险回报比 |
| **均单次收益** | 每笔交易的平均收益率 | 单次操作的期望收益 |
| **震仓频率** | 柱排反转次数 / 持仓周数 | 庄家洗盘频率，越高越难持有 |

**评分分级：**

| 月基评分 | 评级 | 说明 |
|:-------:|:----|:------|
| ≥0.8 | 🏆 仁慈级 | 震仓低、持有体验好，优先建仓 |
| ≥0.3 | ✅ 正常级 | 可参与，需关注震仓频率 |
| ≥0 | ⚠️ 震荡级 | 震仓频繁，谨慎持有 |
| <0 | ❌ 凶残级 | 恶庄，回避 |

## 关键发现

1. **ETF/指数真均幅低（2.3~2.5%）**，适合做基准仓位，不适合冲高策略
2. **股票类真均幅（4.5~5.4%）远高于ETF**，冲高策略在个股上收益更大
3. **盘子越小冲高越大**：中证2000(5.39%) > 中证1000(5.22%) > 中证500(4.96%) > 沪深300(4.52%)
4. **非板块(5.38%)和zdata_only(7.37%)最高**——包含小盘/北交所/次新股，波动剧烈
5. **真冲高率92%**意味着几乎任何股票在任何一周都有92%概率盘中冲高

## 多长冲高连续性结论

**下周冲高在连续多长中高度连续，非散落出现。**

| 指标 | 数值 |
|:-----|:----:|
| 多长连续段总数 | 154,936 段 |
| 平均段长 | 7.1 周 |
| 多长期间下周冲高概率 | **93.9%** |
| 无条件基准概率 | 91% |

**多长第N周的下周冲高概率（无衰减）：**

| 多长第N周 | 下周冲高概率 |
|-----------|:----------:|
| 第1~15周 | 93~96%（稳定，无衰减） |

**连续性分析：**

| 指标 | 概率 |
|:-----|:----:|
| 连续两周都下周冲高 | **89.4%** |
| 本周冲高 → 下周继续冲高 | **98.5%** |
| 无条件冲高概率 | 93.9% |

**策略意义：** 多长+冲高连续适合做趋势跟踪波段。一旦进入多长且第一周确认冲高，大概率可持续数周。多长越长，冲高连续性越稳定，不是"进得越久越危险"。

#### 2.1.4 对VBA代码的影响

当前 `IQQQ跨码_D据擎2神谕.bas` 中`_多长`区旧逻辑将基仓/浮仓混在同一个区域内，需重构为独立分层：

```
旧逻辑：
  _多长区 → 基仓(BTZA>0) / 浮仓(BTZA≤0) / 减仓(BTZC≤0) / 清仓(BTZE≤0)

新逻辑：
  _多长区 → 仅基仓（单一仓位，满足条件即持有，不下破DJE不退）
  浮仓   → 独立操作模式，由冲高信号触发，不隶属_多长区
              ├── <周冲策略>：基于周柱排/周波型，DXZA<0或DXAB<0时启用
              └── <日冲策略>：基于日柱排，DXAB>0且DXZA>0时启用

  仓位跟随区划与信号：
    _多长      → 持有基仓
    _多被   → 可参与（视情况开基仓）
    _空看    → 不参与
    _空长   → 不参与
    浮仓信号触发 → 在_多长保护下开浮仓
```

**分层策略决策树（VBA实现）：**

```
                  ┌─ <月基策略> ──→ 长期策略（针对_多长，按WXZC>0持有）
                  │
当前持仓 ──────────┤
                  │          ┌─ 能<日冲策略>？→ <日冲策略>策略
                  └─ 浮仓 ──┤
                            └─ 不能？→ 能<周冲策略>？→ <周冲策略>策略
                                      │
                                      └─ 都不能？→ 平浮仓，回归基仓
```

**验证闭环：** M3回测给出了"理论上三层策略是否有效"的定量答案；M5实时提示给出了"当前持仓应该怎么做"的操作指令。

**待代码实现（优先级排序）：**

2. [ ] 浮仓逻辑独立：冲高信号（下柱冲高提示/日柱排条件）直接触发浮仓入场，不再作为_多长区的子分类
3. [ ] 仓位退出统一由区划切换或信号触发（区划≠_多长时清仓基仓，下破DJC时平浮仓）
