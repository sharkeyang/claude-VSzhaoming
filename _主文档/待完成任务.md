## §8 日周跨周期：WXAB负交→DXAB正交带动转化

**来源：** [研究日级别探索路径.md](研究日级别探索路径.md) §九
**优先级：** 🥇 高
**类型：** 概念讨论（先讨论清楚逻辑框架，再定量回测）

### 核心问题

当周级别 `WXAB < 0`（负交，周线空头主导），但日级别 `DXAB > 0`（正交，日线多头主导）时：

```
日线 DXAB > 0（正交） → 日级别逐步拉升
        ↓
日级别累积 → 逐步带动周级别 → WXAB 由负趋零 → 最终转正
        ↓
关键问题：这个过程可预测吗？需要什么条件才能成功？
```

### 拆解为三个子问题

| # | 子问题 | 含义 |
|:-:|:-------|:------|
| 1 | **DXAB 正交的持续性** | DXAB>0 能维持多少柱？在转负之前，能否累积足够的日线涨幅来拉动 WXAB？ |
| 2 | **WXAB 转正的阻力** | WXAB<0 的负交有多大"惯性"？DXAB 正交需要多久/多强的日线行情才能抵消？ |
| 3 | **边界条件** | 什么情况下 DXAB 正交也无法拉动 WXAB（失败条件）？什么情况下确定性最高？ |

### 预期讨论产出

- [ ] WXAB 负交→DXAB 正交的"带动链条"：日线 → 周线的传导机制描述
- [ ] 成功转型的条件清单（需要 DXAB>0 持续 N 柱？需要日均涨幅 >X%？）
- [ ] 失败模式识别（DXAB 正交但 WXAB 持续负交不动的特征？）
- [ ] 实操规则：此场景下是否值得试仓？什么位置入场？止损怎么设？
- [ ] 量化回测计划：定义条件 → 回测历史数据 → 验证带动成功率

### 与现有系统的关系

```
周级别框架：
┌─────────────────────────────────────────────┐
│ WXAB < 0（负交）→ 周级别偏向空头             │
│ WXZC > 0（DJC之上）→ 但仍有操作空间           │
└─────────────────────────────────────────────┘
         ↓ ↑ （日线行情积累影响周线方向）
日级别框架：
┌─────────────────────────────────────────────┐
│ DXAB > 0（正交）→ 日级别多头主导              │
│ DXZE > 0（DJE之上）→ 日级别操作许可           │
│ 需要：悬阳升连/突破回踩等日机形态              │
└─────────────────────────────────────────────┘
```

### §7.2 月基WXZA>0延续性分析

月基策略中，当前WXZA>0时，下周能否延续？关键因素：

| 因素 | 最优条件 | 延续率 |
|:----|:---------|:------:|
| **波型** | **龙猪** | **87.7%** |
| **盈提示** | **盈高** | **94.1%** |
| **柱排** | **升排** | **89.1%** |
| **ZA区间** | ZA3~5 | 84.0% |
| **最佳组合** | **龙猪+升排+ZA1~5** | **92.8%** |

**龙猪波型的WXZA>0延续率最高（87.7%），远优于震正（71.3%）。**

**盈高是延续性的最强单维度指标（94.1%），有盈高时下周几乎必定继续站稳A线。**

**组合策略建议：** 月基选股优先筛选龙猪波型+升排+ZA1~5，WXZA>0延续率达92.8%。
