## 第一章 金条件策略

## 1.1 设计哲学 + 梯度分析

### 核心问题
> 是否只使用最优条件进行下周冲高判定？次要条件是否有分数？

### 答案

**周冲评分公式：** 四域完整度(20) + ZA区间(30) + 柱排+盈提示(25) + 波型(25) = 100（详见§1.2）

策分卡本质是 **0~100 的连续评分**，不是"通过/不通过"的二元判定。全量梯度分析证实：

| 策分档 | 周数 | 真均幅 | 相对基准 | 操作建议 |
|:------|:----:|:-----:|:--------:|:--------|
| 🏆 85~100 | 254272 | **11.90%** | 2.5x | A级 核心买入 |
| 🟢 70~84 | 648007 | **6.32%** | 1.33x | B级 辅助买入 |
| 🟡 50~69 | 638106 | **5.70%** | 1.20x | C级 观察 |
| 🟠 30~49 | 1703450 | **3.80%** | 0.80x | D级 回避 |
| 🔴 0~29 | 983395 | **2.95%** | 0.62x | E级 剔除 |
| **基准** | **4227230** | **4.76%** | 1x | — |

**关键结论：梯度平滑，无阈值效应。** 70~84 仍有 6.32% 真均幅（高出基准 32%），次优条件有价值。

### 次要条件的自然处理
策分卡的 **加权求和** 设计已自动处理次要条件：

| 场景 | 四域(20) | ZA(30) | 柱排+盈(25) | 波型(25) | 总分 |
|:----|:-------:|:------:|:----------:|:--------:|:---:|
| 全部最优 | 20 | 30 | 25 | 25 | **100** |
| 波型差 | 20 | 30 | 25 | 5 | **80** |
| ZA超买+波型差 | 20 | 10 | 25 | 5 | **60** |
| 四域一般+ZA差 | 5 | 10 | 15 | 10 | **40** |

分数 = 优先级排序，无需额外逻辑。

## 1.3 策分条件演化递进阶梯

从基本多长到最优条件，存在清晰的递进关系：

| 层 | 条件铺叠 | 周数 | 占总 | 占上级 | 真冲高 | HR≥3% | HR≥5% | HR≥7% | 真均幅 |
|:-:|:---------|:----:|:---:|:-----:|:-----:|:-----:|:-----:|:-----:|:------:|
| 0 | 全量(基准) | 4227230 | 100% | — | 91.5% | 50.6% | 32.8% | 21.9% | 4.76% |
| 1 | 四域=多长 | 1221730 | 28.9% | 28.9% | 95.5% | 63.4% | 45.3% | 32.6% | 6.57% |

**关键洞察：**
- 最大跳变在"升排+非孕"（+柱排稳定性），而非ZA区间

- 盈提示是爆发力放大器（+盈后真均幅从9.58%→23.38%）

## 1.4 条件筛选意义解释

> 每个筛选条件都有明确的VBA定义和实际意义，以下是逐条解释。

<table border="1" cellpadding="8" cellspacing="0" style="border-collapse:collapse;border:2px solid #888;margin:8px 0"><tr><td>

**1. 四域完整度（20分）——周线趋势在哪个区域？**

**VBA判定：**
```
' 四域判定逻辑（基于位谕of周层大局 + 位谕of周层护型）
If WXZC > 0 And DXZE > 0 Then  ' WJC之上
    If WXCD = '金' And (WXAB = '甲' Or WXAB = '乙' Or WXAB = '己') Then
        四域 = '多长'
    ElseIf WXCD = '金' Then
        四域 = '多被'
    End If
Else
    四域 = '空看' Or '空长'
End If
```

**通俗解释：** 趋势好不好？多长=在高速公路上开车，又快又稳。多被=在国道上开，也能走但要小心。空看/空长=在水沟或悬崖边开，不应该参与。

**20分的意思：** 四域是周冲策略的基础条件——只有多长区才能参与。给20分是因为它是入场门槛，没有这个条件其他都免谈。

</td></tr></table>

<table border="1" cellpadding="8" cellspacing="0" style="border-collapse:collapse;border:2px solid #888;margin:8px 0"><tr><td>

**2. ZA区间（30分）——周线站稳均线多少天？**

**VBA判定：**
```
' ZA周 = 周线站稳WJA（A线）的天数（位谕of周类BTZA）
ZA周 = 谕组(X, 位谕of周类BTZA)  ' 取值范围：0~20+
' 赋分逻辑：
If 5 < ZA周 <= 10 Then 得分 = 30    ' 最佳区间
ElseIf 3 <= ZA周 <= 5 Or ZA周 > 10 Then 得分 = 20
ElseIf 1 <= ZA周 < 3 Then 得分 = 10
Else 得分 = 0
```

**通俗解释：** 进场时机对不对？ZA=站稳均线的天数。ZA 5~10=列车刚启动正在加速，是最佳上车窗口。ZA<3=刚启动还不稳。ZA>10=已经涨了一段了，可能接近尾声。

**30分的意思：** ZA是四维度中区分度最高的，因为它直接告诉你当前处于趋势的哪个阶段。

</td></tr></table>

<table border="1" cellpadding="8" cellspacing="0" style="border-collapse:collapse;border:2px solid #888;margin:8px 0"><tr><td>

**3. 柱排+盈提示（25分）——周K线方向+盈利信号？**

**VBA判定：**
```
' 柱排周 = 周K线的排列方向（位谕of周层柱排）
' 盈提示 = 系统计算的盈利信号（位谕of周层盈提示）
柱排周 = 谕组(X, 位谕of周层柱排)  ' 升/跌/人
盈提示 = 谕组(X, 位谕of周层盈提示)  ' 高/宽/丘/空
' 赋分逻辑：
If 柱排升 And 非尾反孕 And 盈有 Then 得分 = 25
ElseIf 柱排升 Then 得分 = 15
ElseIf 人排 Then 得分 = 8
ElseIf 柱排跌 Then 得分 = 5
```

**通俗解释：** 短期趋势稳不稳？升排=本周K线向上，非孕=不是末端十字星，盈提示=系统说有盈利机会。三者同时满足=短期趋势向上+不是末端+有盈利信号，是最好的组合。

**25分的意思：** 柱排和盈提示结合能判断短期趋势的稳定性和是否接近末端。升排+非孕+盈有=25分，说明短期趋势最强。

</td></tr></table>

<table border="1" cellpadding="8" cellspacing="0" style="border-collapse:collapse;border:2px solid #888;margin:8px 0"><tr><td>

**4. 波型（25分）——庄家操作风格？**

**VBA判定：**
```
' 波型周 = 周级别的庄家操作模式（位谕of周层波型）
波型周 = 谕组(X, 位谕of周层波型)  ' 龙猪/龙管/头正/震正等
' 赋分逻辑：
If InStr(波型周, '龙猪') > 0 Or InStr(波型周, '龙管') > 0 Then
    得分 = 25  ' 趋势型庄家
ElseIf InStr(波型周, '头正') > 0 Then 得分 = 15
ElseIf InStr(波型周, '头负') > 0 Or InStr(波型周, '震负') > 0 Then 得分 = 10
ElseIf InStr(波型周, '震正') > 0 Then 得分 = 0  ' 直接否决
```

**通俗解释：** 庄家手法凶不凶？龙猪/龙管=趋势型庄家，走势稳健给上车机会。头正=头部形成可能见顶。震正=震荡型庄家洗盘，下周收益几乎为零（+0.02%）。头负/震负=弱势不参与。

**25分的意思：** 波型反映庄家风格。龙猪/龙管给满分，因为趋势型庄家最友善。震正给0分直接否决。

</td></tr></table>

<table border="1" cellpadding="6" cellspacing="0" style="border-collapse:collapse;border:2px solid #f44;margin:8px 0"><tr><td>

**5. 震正波型直接否决**


**震正波型直接否决，得0分，不参与周冲策略。**

震正波型代表庄家在高位震荡洗盘，下周基本没有上涨动力。
数据证实：震正波型下周收益仅+0.02%，几乎等于随机。
出现震正波型的股票，无论其他条件多好，都不参与周冲策略。

## 1.6 操作流程 + 连续性数据

### 连续性数据

| 指标 | 数值 |
|:----|:----:|
| 多长平均段长 | **7.1 周** |
| 多长期间下周冲高概率 | **93.9%（稳定，15周无衰减）** |
| 连续两周都下周冲高 | **89.4%** |
| 本周冲高 → 下周继续冲高 | **98.5%** |

### 操作频率：低

一个多长段(7.1周)只需 **2次操作**（买入+卖出），中间每周检查信号但不操作。

```
┌─── 多长段开始 ────────────── 多长段结束 ──┐
│ [买入]─持有─持有─持有─持有─持有─持有─[卖出] │
│  ↑策分>=85        中间检查      ↑四域变空   │
│  1次买          不动          1次卖         │
└────────────────────────────────────────────┘
```

### 操作规则

| 事件 | 操作 | 理由 |
|:----|:----|:------|
| **四域进入多长 + 策分>=85** | **买入** | 最优入场点 |

## 1.8 策分卡七大结论 + 核心结论

### 1. 体积效应：市值越小冲高越大

任何股票在任何一周都有92%概率盘中冲高。这不是策略的功劳，是市场规律。

### 3. 多长区是唯一有效操作区

### 5. 盈提示是正信号，不是见顶信号
