## 第一部分：月基选股——股性判断（仁庄 vs 恶庄）

## 1.0 设计初衷：从股性到策分

### 1.0.1 股性的本质

> **股性** = 一只股票在四域操盘法下的**持仓收益特征**。
>
> 具体做法：多长介入 → 下破WGC退出 → 统计该段累计收益 → 跑遍该股所有时段 → 得到全生命周期累计收益。
>
> 这个累计收益就是该股票的**股性分**——它量化了"跟这个庄家玩，长期能赚多少钱"。

### 1.0.2 股性两极画像

| 特征 | 🏆 仁慈庄家 | 💀 恶庄 |
|:----|:-----------|:--------|
| 上升 | 缓慢平稳拉升，回撤小 | 波折多、震仓频繁，非常缓慢 |
| 退出 | 缓慢下破均线，给充足退出时间 | 一个大阴柱直接砸破底线 |
| 收益 | 累计收益高（正分） | 累计收益低甚至为负（负分） |
| 结论 | **优先选择**：每日操作首选标的 | **直接屏蔽**：不跟恶庄玩 |

### 1.0.3 核心设计问题与解答

**Q1：不同上市时长的股票如何可比？**

> 用 **年化收益率** ≈ 累计收益 ÷ 存续时间。上市早的累计收益天然高，除以时间后归一化，使新股老股在同一尺度上比较。

**Q2：上市后一直下跌的股票会怎样？**

> **不会受害。** 一直下跌的股票从未进入 _多长 区，因此没有持仓段，得分为**空（无分数）**而非低分。它最多是"不配参与"，而不是"参与了亏钱"——正是用户说的"不存在于主升期收割，最多没有分数，不是分数很低"。

**Q3：恶庄股会得到什么分数？**

> 频繁拉升→下破WGC→再拉升→再下破的恶庄股，多长段极短、每次收益低甚至为负，累计收益可能为**负数**。这类股票正是月基策分要**屏蔽**的对象。

**Q4：月基策分和周冲择时的关系是什么？**

> 月基策分回答 **"这股票值不值得玩"**（选股），周冲策分回答 **"这周能不能买"**（择时）。
> 股性好的股票优先作为每日操作的备选标的，但具体买点仍需周冲/日冲策分决定。

### 1.0.4 从股性到月基策分的推导

股性分（原始） = Σ(每段多长收益) / 存续时间

但仅有累计收益不够，还需考虑**持仓体验**：

| 问题 | 惩罚因子 | 公式体现 |
|:----|:--------|:---------|
| 震仓多容易拿不住 | 震仓频率 | 分母 (1+震仓频率) |
| 单次收益薄没意义 | 均单次收益 | 乘法因子 |
| 胜率低心态不稳 | 胜率 | 乘法因子 |
| 盈亏比低不划算 | 盈亏比 | 乘法因子 |

**最终公式：**

> **月基策分 = 年化收益 × 胜率 × 盈亏比 × 均单次收益 ÷ (1 + 震仓频率)**

这就是在"原始股性分"基础上，叠加了持仓体验惩罚后的综合分数。分数越高 ≈ 庄家越仁慈 ≈ 越适合长期持有。

### 1.0.5 应用规则

| 分数 | 评级 | 操作 |
|:----:|:----|:-----|
| ≥0.8 | 🏆 仁慈级 | 优先建仓，做月基底仓首选 |
| ≥0.3 | ✅ 正常级 | 可参与，关注震仓频率 |
| ≥0 | ⚠️ 震荡级 | 谨慎持有，恶庄多 |
| <0 | ❌ 凶残级 | **直接屏蔽，不跟庄玩** |
| 无分数 | — | 从未进入多长区，暂不参与 |

## 1.1 月基评分原则

### 1.1.1 设计原理

月基策分的核心目标是评估一只股票在月基策略（多长介入→WXZC<0退出）下的持仓体验。

**为什么用这个公式？**

| 维度 | 含义 | 高值特征 | 低值特征 |
|:----|:-----|:--------|:--------|
| **年化收益** | 长期持有年化收益率 | 趋势强劲，持续上涨 | 趋势弱，震荡不前 |
| **胜率** | 盈利交易占比 | 多长段内稳定上涨 | 多长段内频繁回调 |
| **盈亏比** | 盈利/亏损比值 | 上涨幅度远大于下跌 | 下跌幅度接近上涨 |
| **均单次收益** | 每笔收益率 | 单次趋势利润丰厚 | 单次利润薄 |
| **震仓频率** | 柱排反转/持仓周数 | 庄家仁慈，趋势稳定 | 恶庄频繁洗盘 |

**核心逻辑：** 高分=多长段内趋势稳定+震仓少+退出平缓。这对应的是"仁慈庄家"特征：
- 上涨时回撤小，持股体验好
- 退出时缓慢下破均线，给充足退出时间
- 柱排方向稳定，不频繁反转
- 不挂山顶，趋势末端不出现极端冲高后暴跌

### 1.1.2 分数分布统计（全量7463只）

| 区间 | 评级 | 只数 | 占比 |
|:----:|:----|:---:|:---:|
| <0 | ❌ 凶残级 | 0 | 0.0% |
| 0~0.3 | ⚠️ 震荡级 | 7091 | 95.0% |
| 0.3~0.8 | ✅ 正常级 | 96 | 1.3% |
| ≥0.8 | 🏆 仁慈级 | 159 | 2.1% |
| 无数据 | — | 117 | 1.6% |

**月基策分范围：** 0.0000 ~ 144.0736，均值 0.1869

> **注意：** 95.0%的股票集中在0~0.3区间（震荡级），说明绝大多数股票不适合长期持有策略。只有前3.4%的股票（正常级+仁慈级）才适合月基策略。另有117只无数据（多为北交所次新股，历史不足）。


月基评分用于评估一只股票是否适合长期持有（月基策略），分数越高越适合。

**评分公式：**



| 维度 | 说明 | 意义 |
|:----|:-----|:-----|
| **年化收益** | 多长→WXZC<0退出期间的年化收益率 | 策略整体盈利能力 |
| **胜率** | 盈利交易次数 / 总交易次数 | 策略的稳定性 |
| **盈亏比** | 平均盈利 / 平均亏损 | 策略的风险回报比 |
| **均单次收益** | 每笔交易的平均收益率 | 单次操作的期望收益 |
| **震仓频率** | 柱排反转次数 / 持仓周数 | 庄家洗盘频率，越高越难持有 |

**评分分级：**

| 月基评分 | 评级 | 说明 |
|:-------:|:----|:------|
| ≥0.8 | 🏆 仁慈级 | 震仓低、持有体验好，优先建仓 |
| ≥0.3 | ✅ 正常级 | 可参与，需关注震仓频率 |
| ≥0 | ⚠️ 震荡级 | 震仓频繁，谨慎持有 |
| <0 | ❌ 凶残级 | 恶庄，回避 |


## 1.2 设计哲学

月基策分的核心目标是评估一只股票在月基策略（多长介入->WXZC<0退出）下的持仓体验。

**为什么用这个公式？**

月基策分 = 年化收益 x 胜率 x 盈亏比 x 均单次收益 / (1 + 震仓频率)

| 维度 | 含义 | 为什么重要 |
|:----|:-----|:----------|
| **年化收益** | 多长段内年化收益率 | 衡量趋势盈利能力，收益越高分越高 |
| **胜率** | 盈利交易占比 | 胜率越高，策略越稳定，持股体验越好 |
| **盈亏比** | 平均盈利/平均亏损 | 比值越高，说明上涨幅度远大于下跌幅度 |
| **均单次收益** | 每笔交易平均收益率 | 单次期望收益，越高越好 |
| **震仓频率** | 柱排反转次数/持仓周数 | 越低越好，说明庄家仁慈、趋势稳定 |

**核心逻辑：** 高分=多长段内趋势稳定+震仓少+退出平缓。对应"仁慈庄家"特征：
- 上涨时回撤小，持股体验好
- 退出时缓慢下破均线，给充足退出时间
- 柱排方向稳定，不频繁反转
- 不挂山顶，趋势末端不出现极端冲高后暴跌


## 1.3 条件筛选意义解释

> 月基策分的每个评估维度都有明确的定义和实际意义。

**1. 年化收益——长期持有能赚多少？**

**公式：** 多长段内年化收益率 = (1 + 总收益率)^(52/持仓周数) - 1

**通俗解释：** 年化收益越高，说明这只股票在多头趋势中的上涨动力越强。好比一个人工作能力强，年收入就高。

**2. 胜率——做对的概率多大？**

**公式：** 胜率 = 盈利交易次数 / 总交易次数

**通俗解释：** 胜率越高，说明每次进入多长区后上涨的概率越大。好比一个射手的命中率，越高越可靠。

**3. 盈亏比——赚一次能抵几次亏？**

**公式：** 盈亏比 = 平均盈利 / 平均亏损

**通俗解释：** 盈亏比>1说明赚的时候比亏的时候多。好比打牌，赢一把的钱比输一把的钱多，长期就能赢。

**4. 均单次收益——每笔交易平均赚多少？**

**公式：** 均单次收益 = 累计收益 / 交易次数

**通俗解释：** 每笔交易的平均收益，越高越好。好比每次做生意平均赚的钱。

**5. 震仓频率——庄家洗盘凶不凶？**

**公式：** 震仓频率 = 柱排反转次数 / 持仓周数

**通俗解释：** 震仓频率越低，说明庄家越仁慈，持股体验越好。好比坐车，路越平越舒服。震仓频率高=恶庄频繁洗盘，容易被震下车。

**综合评分逻辑：**

月基策分 = 年化收益 x 胜率 x 盈亏比 x 均单次收益 / (1 + 震仓频率)

分母(1+震仓频率)的作用：震仓频率越高，分母越大，分数越低。这确保了震仓多的股票被降分。

## 1.5.1 ZB周预警信号验证（新数据发现）

基于谕组全量7,463只的ZB周列数据，验证了月基策略的退出预警信号。

### ① WXZB从>0转≤0是最强退出信号

| WXZB状态 | 下周P(≥3%) | 真均幅 | 操作 |
|:---------|:----------:|:------:|:----|
| WXZB>0（正常持有） | 63.5% | 6.45% | ✅ 继续持有 |
| WXZB≤0（已下破） | 38.5% | 3.19% | ⚠️ 明显走弱 |
| **从>0转≤0（刚转负）** | **16.7%** | **1.31%** | 🚨 **立即卖出** |
| 转负当周 | **7.6%** | **0.34%** | 🔴 几乎不涨 |
| 转负后下周 | 27.7% | 2.95% | 观望 |
| 转负后下下周 | 31.5% | 3.09% | 仍在恢复 |

**结论：** WXZB从>0转≤0时，当周P(≥3%)=7.6%，真均幅仅0.34%。**月基持仓中WXZB转负即卖，不犹豫。**

### ② WXZA>0延续性分析

月基策略中，当前WXZA>0（站稳A线）时，下周能否延续？

| 因素 | 最优条件 | 延续率 |
|:----|:---------|:------:|
| **波型** | **龙猪** | **87.7%** |
| **盈提示** | **盈高** | **94.1%** |
| **柱排** | **升排** | **89.1%** |
| **ZA区间** | ZA3~5 | 84.0% |
| **最佳组合** | **龙猪+升排+ZA1~5** | **92.8%** |

**结论：** 月基选股优先龙猪波型+升排+ZA1~5，WXZA>0延续率92.8%。盈高是最强单维度（94.1%）。

### ③ 月基策略操作规则（更新版）

```
月基持仓中，每周检查：
  WXZB>0 → 继续持有（下周P(≥3%)=63.5%）
  WXZB从>0转≤0 → 立即退出（当周P(≥3%)=7.6%）
  WXZB≤0 持续 → 观望，等待重新站稳B线

月基选股优先级：
  🏆 龙猪+升排+ZA1~5 → WXZA>0延续率92.8%
  ✅ 盈高 → 延续率94.1%
  ❌ 震正 → 延续率71.3%，尽量回避
```

## 1.8 月基持仓四分类——持有状态判定

> 月基策略的持仓状态应分为四类，每类的5周内WXZB<0概率差异显著。

基于全量7,463只×422万周数据，验证结论如下：

| 类别 | 样本 | 5周续持率 | 5周转负率 | 占比 |
|:----|:---:|:---------:|:---------:|:----:|
| **①退出区(WXZB≤0)** | 216万 | **6.2%** | 93.8% | 51.7% |
| **②消极区(最后一柱信号)** | 103万 | **43.1%** | **56.9%** | 24.6% |
| **③积极区(金+龙猪+升排)** | 42万 | **71.8%** | 28.2% | 10.0% |
| **④不确定区(其他)** | 58万 | **59.4%** | 40.6% | 13.8% |

**分类之间的差距：** 积极vs消极=差28.7pp | 消极vs不确定=差16.3pp | 积极vs不确定=差12.4pp

### 1.8.1 分类判定流程

从WXZB>0开始，优先判定积极条件，再从剩余中逐步筛选消极条件：

```
WXZB≤0? → 是 → ①退出区（93.8%续持率仅6.2%）
否 ↓
金+龙猪+升排? → 是 → ③积极区（71.8%续持）
否 ↓
逐层检查消极信号（触发任一条即归入消极）：
  ├── 跌吞(阴吞前阳)？→ 72.1%转负
  ├── ZA较前周下降？→ 69.3%转负
  ├── 本周HR<0？→ 63.1%转负
  ├── ZC≤0(C线失守)？→ 60.8%转负
  └── 震正波型？→ 54.5%转负
  任一是 → ②消极区（56.9%转负）
  全否 → ④不确定区（40.6%转负，中性）
```

### 1.8.2 各类信号详解

**① 退出信号：WXZB≤0**
- 5周续持率仅6.2%，几乎不可能回到WXZB>0
- 当周P(≥3%)=7.6%，当月基策略退出条件触发时，直接退出

**② 消极信号（最后一柱）——筛选链路（从强到弱）：**

| 信号 | 5周转负率 | 含义 | 操作 |
|:----|:---------:|:-----|:-----|
| 🚨 跌吞(阴吞前阳) | 72.1% | 阴线吞噬前阳，主力已离场 | 出现即准备减仓 |
| 🚨 ZA较前周下降 | 69.3% | 站稳A线能力减弱 | 连续2周下降立即行动 |
| 🚨 本周HR<0 | 63.1% | 周线下跌 | 双重确认时减仓 |
| 🚨 ZC≤0 | 60.8% | C线失守 | 比ZB≤0更严重 |
| 🚨 震正波型 | 54.5% | 趋势动荡不稳 | 龙猪转震正时警惕 |

**③ 积极信号 —— 金+龙猪+升排**
- 金系趋势 + 龙猪趋势庄家 + 升排方向稳定
- 5周续持率71.8%，真均幅11.17%
- 详见：**`MC3.1_研究月基策略最后一柱.md`** §积极条件

**④ 不确定区**
- 无明确消极信号，也无积极条件
- 5周转负率40.6%，接近中性
- 建议：叠加ZA区间/盈提示进一步筛选

### 1.8.3 操作规则

```
月基持仓中，每周检查判定类别：

③积极区 → 继续持有，下周P(≥3%)=61.5%
④不确定区 → 可持有，但关注ZA趋势方向
②消极区 → 准备减仓：ZA下降→关注, 跌吞/HR<0→准备, 双重确认→立即减仓
①退出区(WXZB从>0转≤0) → 立即卖出（当周P≥3%=7.6%）
```

## 1.9 月基操作规则（完整版）

整合选股+持仓四分类+退出三个阶段的完整规则：

### 选股阶段（月基策分）
| 月基策分 | 评级 | 操作 |
|:--------:|:----|:------|
| ≥0.8 | 🏆 仁慈级 | 优先建仓，做月基底仓首选 |
| 0.3~0.8 | ✅ 正常级 | 可参与，关注震仓频率 |
| 0~0.3 | ⚠️ 震荡级 | 谨慎持有，恶庄多 |
| <0 | ❌ 凶残级 | 直接屏蔽 |

选股优先条件：龙猪波型+升排+ZA1~5（WXZA>0延续率92.8%）

### 持仓阶段（四分类检查）

每周按判定流程检查，确定当前状态：

| 类别 | 判定条件 | 5周续持率 | 操作 |
|:----|:---------|:---------:|:------|
| **③积极区** | 金+龙猪+升排 | 71.8% | 继续持有，每周检查 |
| **④不确定区** | WXZB>0，无积极/消极信号 | 59.4% | 可持有，关注ZA趋势 |
| **②消极区** | 跌吞/ZA下降/HR<0/ZC≤0/震正 | 43.1% | 准备减仓 |
| **①退出区** | WXZB≤0 | 6.2% | 立即卖出 |

### 消极区内部操作细则

| 信号 | 操作 | 说明 |
|:----|:------|:------|
| ZA下降 | 关注减仓 | 连续2周下降立即升级 |
| 跌吞 | 准备减仓 | 72.1%概率5周内转负 |
| HR<0 | 准备减仓 | 下周继续恶化立即行动 |
| ZC≤0 | 准备减仓 | C线失守，比ZB≤0更严重 |
| 震正波型 | 警惕 | 龙猪转震正时趋势不稳 |
| 双重确认(ZA下降+HR<0) | **立即减仓** | 不等WXZB转负 |

### 退出阶段（WXZC<0）
- WXZC<0 = 月基策略的最终退出条件
- 消极区预警通常比WXZC<0提前2~5周触发
- 预警系统的作用是**提前退出，不等WXZC<0**
