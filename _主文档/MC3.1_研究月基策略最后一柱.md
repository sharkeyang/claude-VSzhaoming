# 最后一柱特征总结

> 备查文件：利用观察图形，凭直觉发现的特征，后续通过数据回测进行确认与排除，最终形成有效的指标。
> 关联：`全局_C昭明路线图.md §2.2.8`

---

## 核心定义

**最后一柱**：虽然符合WJA之上升排，但已代表主升结束。

- **延续性**：只要不是最后一柱，且处于主升区域，都可备选进行下柱冲高
- **中止性**：主升遇到最后一柱，代表前一波主升趋势的停止，其后转入震荡或开启下一波主升
- **注意**：不一定是最高点，而是**主升浪的结束**

---

## 三大类形态

### 一、逐步放大模式

| 特征 | 说明 | 操作 |
|:-----|:-----|:-----|
| 连阳之后 + 巨阳柱 | 可止盈并不再持仓 | 巨阳(>20) / 巨阳后走平(小阳/十字/小阴) |
| (连阳3+)(孕阴)(大阳) | 连阳后的阴柱，最后冲高，预示最后冲高后结束主升 | 止盈 |
| 连阳之后 + 逐步放大 | 上冲过猛，可能转为锐角下跌 | 止盈 |
| 连阳之后 / 上影线小柱 | 可止盈并不再持仓，代表趋势衰竭 | 止盈 |
| 连阳之后 + 高开悬阴柱 | 可获利并不再持仓。实际如果是孕阴，很可能下柱冲高 | 获利止盈 |
| (中阳)(十字星)(中阳) | 其后可能形成阴柱 | 观察 |

### 二、逐步缩小模式

| 特征 | 说明 | 操作 |
|:-----|:-----|:-----|
| 连阳之后 + 小阳柱/十字星/小阴柱 | 连阳柱体逐步缩小，代表升势衰竭 | 止盈 |
| 连阳之后 + 长下影线小柱（下影线包含于前柱内部） | 很可能属于趋势尾部，开始下跌 | 止盈 |
| (中阳+十字星+中阳) + 长下影小柱 | 下探后恢复，但主力很可能已逃跑，进入震荡 | 观望 |
| 连阳之后 + 先上影再下影 | 代表升势衰竭 | 止盈 |

### 三、由阳转阴模式

代表原升势停止。

| 特征 | 说明 | 操作 |
|:-----|:-----|:-----|
| 由连阳逐步转为阴阳相间 | 当单阳或连阳夹在两个阴柱之间，可能表示已见顶，即将转跌 | 止盈 |
| 连阳后 + 跌孕 + 升吞 + 单阴（/跌孕/跌吞） | 见顶转跌 | 清仓 |
| 连阳后 + 跌孕 + 连阳 + 单阴（/跌孕/跌吞） | 见顶转跌 | 清仓 |
| 跌吞：连阳之后 + 单巨阴柱下破WJA | 代表趋势完全逆转 | 清仓 |
| 久丘后四柱栅：连阳之后出现跌连 | 必然是升势终结 | 清仓 |

## 栏栅调整后的失败情况

栏栅的下柱都会冲高，但（是否收阳柱）（是否后续为连阳）不一定成立。

| 模式 | 后续 |
|:-----|:-----|
| 跌孕 + 升吞/升孕 + 跌吞/跌孕 | (阴+阳+阴)后可能转为下跌 |
| 跌吞 + 升吞/升孕 + 跌孕 | (阴+阳+阴)随后可能转向下跌 |
| (跌吞+升孕) | 可能转为下跌趋势 |

---

## 跌连规律

| 情况 | 概率 | 操作 |
|:-----|:----:|:-----|
| 随后立刻进入连阳 | 极少 | 例外 |
| 由主升转为震荡或下跌 | **多数** | 等待出现(升连)或(启动结构)再介入 |

---

## 三连跌

| 场景 | 判断 |
|:-----|:-----|
| 一段拉升后，出现三连跌后下破WJA | 很可能已逆转为跌势 |
| 一段拉升后，出现三连跌后，出现单阳柱或多个小阳柱反弹，再次下破WJA | 很可能已逆转为跌势 |

---

## 根（WJA之下）

暂下破WJA：

- 由于WJA压制导致下柱高幅较小
- 针对"已冲高""已整弧"，下破WJA后很可能快速下跌
- **WTZA因素**：WTZA=-1首下破，有可能会再次站上WJA；针对WTZA≤-2，已沿WJA形成趋势

---

## 枝（突破WJA后的失败情况）

单鼎上破WJA——特别是当WJA与WJB纠缠时，很可能失败。
要等待第二柱确认（回踩/升连）→ 要明确突破WJA很可能是假突破。

---

## 升排失势的特征

| 特征 | 说明 |
|:-----|:-----|
| 震荡后的升排 | 前期包括冲高回落至WJA，长上影等情况，出现升排失势的反例。配合日类可见主升已结束 |
| 大阳后大阴回到WJA，继续连阳，接近前高 | 顶部 |
| (长上影阳柱)后连阳，接近前高 | 顶部 |
| 连阳后(长上影) | 代表不再紧贴上沿，很可能脱轨 |
| 连阳后(长下影) | 代表已出现离场仓位，后力不足 |

## 待验证项

> 以上特征均为直觉观察，需通过数据回测确认：
> 1. 逐步放大模式中各类形态的后续走势概率
> 2. 逐步缩小模式中各类形态的后续走势概率
> 3. 由阳转阴模式中各类形态的后续走势概率
> 4. 跌连后立刻连阳 vs 转震荡的概率统计
> 5. 三连跌后逆转为跌势的概率
> 6. 升排失势各特征的量化指标

## 数据验证结论（2026-07-17，全量7463只）

基于谕组全量数据的验证结果——柱排形态 → 5周内WXZB<0的概率：

| 特征 | 样本 | 5周内WXZB<0 | 预警强度 |
|:----|:----:|:----------:|:--------:|
| 🚨 **跌吞(阴吞前阳)** | 15,963 | **72.1%** | **最强** |
| ⚠️ 单柱由阳转阴 | 320,094 | 55.8% | 强 |
| ⚠️ 连阳3+转阴 | 123,411 | 45.3% | 中 |
| ⚠️ 连阳3+十字星 | 87,010 | 42.7% | 中 |
| ▫️ 连阳逐步缩小 | 9,075 | 38.0% | 弱 |
| ▫️ 连阳3+巨阳>10% | 34,364 | 19.7% | 安全 |
| ▫️ 连阳3+巨阳>20% | 8,863 | **14.2%** | 最安全 |

**核心发现：**
1. **跌吞是5周内WXZB<0的最强预测信号（72.1%）**——出现跌吞，即使WXZB仍然>0，也应在5周内准备退出
2. **巨阳柱不代表见顶**——连阳3+巨阳>20%后，5周内转负率仅14.2%，趋势仍然强劲
3. **由阳转阴需要叠加ZA下降才能确认**——单柱由阳转阴转负率55.8%，若同时ZA下降则升至69.3%
4. **连阳逐步缩小转负率仅38.0%**——缩小不代表立即转跌，很多是正常整理

**与应用：**
- 月基持仓预警系统中新增**跌吞检测**——优先级仅次于ZA趋势下降
- 跌吞+ZA下降双重确认 → 立即减仓（不必等WXZB转负）
