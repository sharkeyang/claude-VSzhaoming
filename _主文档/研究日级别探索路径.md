## 一、探索背景

### 1.1 为什么要探索日级别

现有回测数据显示日级别方向预测力有限（各类条件下胜率47%~52%），**核心价值在筛选冲高幅度而非方向**。但日级别作为操作落地的最小时间尺度，以下场景仍需要日级别判断：

| 场景 | 日级别的作用 |
|:----|:------------|
| 日浮仓策略 | 尾盘判断次日冲高概率，决定是否开日浮仓 |
| 周浮仓的日级别择时 | 周级别方向确定后，日级别决定具体入场点 |
| 持仓过程中的日级别管理 | 日级别阴柱/阳柱的应对（加浮仓/持有/减仓） |
| 特殊形态识别 | 鼎/薡/栅等日级别特殊形态的独立识别价值 |

### 1.2 探索方法

每个探索路径采用以下流程：

```
① 形态定义 → ② 条件量化为VBA指标 → ③ 回测验证 → ④ 确认有效/无效 → ⑤ 纳入规则或废弃
```

### 1.3 总体框架

```
日级别探索
├── 一、基本条件（什么位置会有阳柱）
├── 二、最有把握日机（管中稳定）
├── 三、突破回踩
├── 四、其他日机（顺势/捡漏/升排）
├── 五、日警（偏BSHA/偏管/管宽5-15）
├── 六、筛选日类机会
├── 七、特殊形态汇总（鼎/薡/栅/管中阴/D1b）
└── 八、向上突破/向下突破
```

## 二、基本条件：日类什么位置会有阳柱

> **前提：** 所有日级别操作都必须在周级别条件满足的前提下进行（`WXZC>0` 且 `DXZE>0`）

### 2.1 硬条件

| # | 条件 | 说明 |
|:-:|:-----|:-----|
| 1 | `收盘价 > DJC` | DJC之上是最低要求 |
| 2 | `DXCD > 0` | 日级别护级为正 |
| 3 | 周级别：`WXZC > 0` | 前周WXZC大于0方可操作 |

### 2.2 趋势背景分类

不同趋势背景下，日级别阳柱的概率不同：

| 趋势背景 | 阳柱概率 | 说明 |
|:--------|:--------:|:-----|
| 上升大趋势中已长久下跌后，刚转为上升 | 🟢 高 | 底部反转后的初期上升 |
| 上升途中忽然下跌（合顶情况下忽然阴柱） | 🟡 中 | 上升中继回调，需区分是否诱空 |
| 主升浪中持续上涨 | 🟢 高 | 趋势延续 |
| 主升浪末期/顶部震荡 | 🔴 低 | 见顶信号增多 |

### 2.3 探索问题

- [ ] 在DJC之上，不同趋势背景下阳柱/冲高概率统计
- [ ] DJC之上 vs DJE之上 vs DJA之上的阳柱概率差异
- [ ] 合顶情况下忽然阴柱→后续阳柱的概率

## 三、探索路径一：最有把握日机 — 管中稳定

### 3.1 窄管中蓄力

**核心逻辑：** 在DJA附近低波动窄幅震荡，均线之上积蓄力量。

**条件：**

| 条件 | 要求 | 指标 |
|:----|:-----|:-----|
| 管宽 | `DS顶JA ≤ 8` | 顶JA管宽度量 |
| 波动 | 低波动维持较长时间 | 柱幅标准差 |
| 均线位置 | 在均线之上 | 收盘价 > DJA 且 > DJB |

**探索子项：**

- [ ] 管宽≤8时阳柱概率 vs 管宽>8时
- [ ] 低波动持续时间对阳柱概率的影响（持续时间越长越好？）
- [ ] 均线之上 vs 均线之下的蓄力效果差异

### 3.2 窄管中多柱形态

#### 3.2a 最佳形态：连升且连触

**条件：**
- 柱排为升连（连续阳柱）
- 连续上触管壁（连触）
- 可出现跌孕（升连后跌孕）

#### 3.2b 次佳形态：调整结构

**条件：**
- 启动结构（上破DJA后的初期调整）
- 栏（横向整理）
- 栅（三柱/四柱组成的整理形态）

#### 3.2c 可接受的阴柱形态

**条件：**
- 阴柱类型：管中/跌孕/跌吞/跌连
- 未下破DJA 或 首下破DJA
- 阴柱后需有止跌信号

**探索子项：**

- [ ] 连升且连触→后续冲高概率统计
- [ ] 启动结构/栏/栅的识别与胜率
- [ ] 管中阴柱未下破DJA vs 下破DJA的后续差异
- [ ] 首下破DJA（第一次跌破）的介入价值

### 3.3 管中多次反弹

**核心逻辑：** 在下壁止跌后向上壁移动过程中（向上但未触上壁）。

**条件：**

| 条件 | 要求 |
|:----|:-----|
| 管中位置 | 下壁止跌后向中上壁移动 |
| 方向 | 向上但未触上壁 |
| DJA | 在DJB之上 |
| DTZA | `0 < DTZA ≤ 6` |
| 柱排 | 非跌排 |
| 连阳 | 连阳≤3，最好为跌孕 |

**探索子项：**

- [ ] 下壁反弹→中上壁移动→冲高概率
- [ ] DTZA在(0,6]区间内不同值的胜率分布
- [ ] 连阳≤3的约束是否有效
- [ ] 跌孕作为买入信号的胜率

## 四、探索路径二：突破回踩

### 4.1 基本突破回踩

**条件：**
- 刚突破DJA（价格由下向上穿越DJA）
- 连阳后跌孕（突破以连阳完成，随后回调跌孕）
- 既是升排，又有DJA支撑

### 4.2 突破后的启动结构

**条件：**
- 第一柱：上破DJA（阳柱突破）
- 第二柱：跌孕（回踩确认）
- 第三柱：继鼎（继续上升）

### 4.3 潜在四柱栅

**条件：**
- DJA之上
- 中阳后跌孕
- 跌孕后小阳
- 构成四柱栅形态

### 4.4 悬阳升连

**条件：**
- `DXAB` 正交
- 从DJA之下以阳柱上破DJA
- 在DJA之上出现悬空/十字星/小阳柱/小阴柱
- 参考：宁波方正@20250518

**探索子项：**

- [ ] 突破回踩各子形态的胜率对比
- [ ] 启动结构中继鼎的成功率
- [ ] 四柱栅的识别规则与胜率
- [ ] 悬阳升连的识别与验证
- [ ] 突破后DTZA值对后续走势的影响

## 五、探索路径三：其他日机

### 5.1 【顺势】触顶后未冲高

**条件：**
- 触顶（触及管壁上沿）或触哼
- 未冲高（未出现大幅冲高后的回落）
- 管哼DJA保持宽固定且小于6

### 5.2 【捡漏】触b

**场景一：阴柱止跌于DJA**
- 价格触b
- 阴柱在DJA处止跌
- 阴柱后出现十字星或小阳柱（止跌信号）

**场景二：初始突破DJC后的合顶回调**
- 初始突破DJC
- 合顶
- 触a或触b
- 跌连下破DJA
- 需要等待重新站上DJA

### 5.3 【寻找升排】在DJC之上寻找升排片段

**核心逻辑：** 在DJC之上分布间断的升排片段，要在以下位置寻找：

| 位置 | 条件 |
|:----|:-----|
| 刚出现升排 | 升排初现，具有延续性 |
| 单阳柱之后 | 单阳柱可能开启升排 |
| 人排或跌排之后 | 人排/跌排结束后止跌信号 |

**基本要求：** 站上DJA

**探索子项：**

- [ ] 触顶未冲高→后续冲高概率
- [ ] 触b止跌信号的可靠性
- [ ] 升排片段在不同位置的持续概率
- [ ] 单阳柱→升排的转化率

## 六、探索路径四：日警（风险预警）

### 6.1 偏BSHA 警告

**规则：** 不要相信偏离DJA较大的阳柱。

> 当阳柱偏离DJA幅度过大时，即使形态上看是强势突破，也容易回落。

**量化条件：**
- `DSHA`（偏离DJA的幅度）大于某一阈值
- 阳柱虽然强势，但偏离过大导致回撤风险高

**探索子项：**

- [ ] DSHA阈值多少时阳柱后续冲高概率显著下降
- [ ] 偏离DJA较大的阳柱 vs 紧贴DJA的阳柱，后续冲高概率对比

### 6.2 偏管警告

**规则：** 去掉偏离DJC>20%的DTZA=1/2。

> 当价格偏离DJC幅度超过20%时，DTZA=1或2的参考价值降低。

**条件：**
- `价格偏离DJC > 20%`
- `DTZA = 1` 或 `DTZA = 2`

### 6.3 管宽(5,15) 在DJA附近找机会

当管宽处于(5,15)时，在DJA附近找机会：

| 规则 | 说明 |
|:----|:-----|
| ❌ 不要相信DTZA过久 | DTZA值过大说明在DJA之上停留太久，动能衰竭 |
| ✅ 可以相信升排 | 升连/升吞/跌孕 → 可靠的上升信号 |
| ✅ 可以相信跌排止跌 | 跌排后止跌（十字星/小阳柱）→ 下跌衰竭信号 |
| ❌ 不要相信DTZA=1 | 要等待DTZA两柱以上，单柱站上DJA不可靠 |
| ✅ 可以相信薡/DTZA=-1/-2 | 已冲高应止盈，未冲高可以试仓 |
| ✅ 阴柱后立刻回升 | 相信阴柱后立刻回升的力度 |

**探索子项：**

- [ ] DTZA过久的阈值定义（>多少算过久？）
- [ ] 管宽(5,15)区间内不同DTZA值的胜率分布
- [ ] 升排（升连/升吞/跌孕）在管宽(5,15)下的胜率
- [ ] 跌排止跌信号的可靠性
- [ ] DTZA=1 vs DTZA≥2的可靠性对比
- [ ] 薡/DTZA=-1/-2的试仓策略胜率

## 八、探索路径六：特殊形态汇总

### 8.1 鼎（Ding — 介于DJC丘中间态）

> ⚠️ **重要警告：** 不要过分重视日鼎。鼎是介于DJC丘中间态，是现在主要操盘方法，失败率高，**应该舍弃**鼎作为主要决策依据。

#### 鼎分类（按并符划分）

**分类标准：** 不按幅度划分，按并符划分。剔BSHA≥5/涨≤1。

| 类型 | 并符 | 说明 |
|:----|:-----|:-----|
| **继鼎** | 升连 | 多头延续 |
| **继鼎** | 升吞（吞鼎） | 多头吞没 |
| **继鼎** | 升孕（孕鼎） | 前有两阴有可能大跌（注意诱多） |
| **假鼎** | 阴柱后 | 跟在阴柱之后，虽然为鼎但是下跌形态 |
| **首鼎** | 第一柱 | 第一柱未必能站DJA，等待回踩站稳 |

#### 鼎后阴

- 要鼎后5内（鼎出现后5个交易日内出现阴柱）
- 要结合周柱（需结合周级别判断）

#### 对鼎

- 不按幅度划分，按并符划分
- 剔BSHA≥5/涨≤1

#### 鼎的相关规则

**用途建议：** 并符仅是参考信号，不能作为买入决策。应在周类决策满足时参考这些信号。

### 8.2 薡（Ding2 — D2e首下破）

**核心条件：** `D2e` 首下破（第一次跌破D2e线）

**应用场景：**
- 已冲高应止盈
- 未冲高可以试仓
- 相信阴柱后立刻回升

### 8.3 四柱栅 / 三柱栅

#### 初突破DJC时

在初突破DJC时，四柱栅/三柱栅是机会信号。

#### 久运行于DJC之上时

当价格久运行于DJC之上时，四柱栅/三柱栅可能是顶部标志：
- 如多柱远离DJA后以鼎靠近DJA → 此为警诱

#### 兼容情况

| 情况 | 条件 |
|:----|:-----|
| 始终在DJA之上 | 最后柱为中阳 |
| 有一柱下破DJA | 最后为首鼎（仅此情况考虑首鼎） |

### 8.4 管中阴（初）

**条件：**
- 孕阴
- `DTZC ≤ 6`
- `DTZA ≤ 4`

### 8.5 D1b 特殊形态

## 九、探索路径七：向上突破 / 向下突破

### 9.1 【向上突破】

**核心原则：** 找突破之后回踩。

| 回踩信号 | 说明 |
|:---------|:-----|
| 回踩跌孕 | 突破后以跌孕形态回踩，确认支撑 |
| 鼎 | 突破后以鼎形态回踩确认 |

### 9.2 【向下突破】

**策略一：找下跌之后的止跌再升**

| 阶段 | 信号 |
|:----|:-----|
| 止跌 | 阴柱变小 / 升孕 |
| 回升 | 升吞 / 升连 |

**探索子项：**

- [ ] 突破回踩各子形态的盈亏比统计
- [ ] 止跌信号（阴柱变小/升孕）→回升的转化率
- [ ] 首下破D2e→后续走势分布

## 十、数据验证计划

### 10.1 分级验证策略

| 等级 | 范围 | 方法 | 优先级 |
|:----:|:-----|:-----|:------:|
| **P0** | 最有把握日机（管中稳定） | 全量条件回测 | 🥇 最高 |

| 指标 | 含义 | 目标值 |
|:----|:-----|:------:|

| 均幅 | 下柱平均涨幅 | >0.5% |
| 盈亏比 | 盈利交易/亏损交易比值 | >1.2 |
| 样本量 | 符合条件的交易次数 | >100 |
