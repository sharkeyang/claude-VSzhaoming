## 策略有效性系统性验证

> **验证依据：** MC3.8_策略验证loop.py 对谕组全量 7,463 只×4,229,602 周数据的 5 维度系统性检验
> **验证时间：** 2026-07-17

### 为什么要做系统性验证？

策分是一个单一概率值（如"金+升排+非孕 = 77.4%"），但一个数字掩盖了五个重要问题：

| 问题 | 举例 | 影响 |
|:----|:------|:------|
| **① 概率稳定吗？** | 77.4%是50周平均还是5周平均？近10周是否下滑到70%？ | 策分可能已过时 |
| **② 样本够吗？** | 77.4%基于多少周数据？100周 vs 10000周可信度差很远 | 小样本策分不可靠 |
| **③ 尾部风险？** | 即使77.4%达标，那22.6%失败的周里最大跌幅是多少？ | 单次亏损可能吃掉多次收益 |
| **④ 条件交互？** | 升排+非孕效果好，但再加波型=震正是否直接归零？ | 策略叠加可能产生负效应 |
| **⑤ 时间衰减？** | 这个条件3年前有效，现在是否仍然有效？ | 市场风格变化后旧规律失效 |

**核心矛盾：** 策分只能回答"历史上有多少比例冲高"，但无法回答"这个策略未来是否依然可靠"。系统性验证就是要回答后者。

### 如何验证？

5 维度验证框架，每项策略逐一检验：

| 维度 | 方法 | 判定标准 | 
|:----|:------|:---------|
| **① 稳定性** | 按年切片计算 P(≥3%)，观察年际波动幅度 | 最大偏差 ≤5pp = 稳定；≤10pp = 可接受；>10pp = 警告 |
| **② 置信度** | Wilson 95% 置信区间宽度 | 宽度 ≤10pp = 高置信度；≤15pp = 可接受；>15pp = 样本不足 |
| **③ 样本量** | 策略命中总周数 | ≥5000 周 = 可信；500~5000 = 需谨慎；<200 = 不可靠 |
| **④ 尾部风险** | 失败周中最差5%的HR分布 | 最差5% ≤ -5% = 高尾部风险 |
| **⑤ 时间衰减** | 近年(2023-2026)趋势是否与历史均值方向一致 | 逐年显著下降 = 策略可能失效 |

### 验证结论

**10 个策略按可靠性分为三级：**

| 级别 | 策略 | P(≥3%) | 样本 | 稳定性 | 尾部风险 | 时间衰减 |
|:----:|:-----|:------:|:----:|:------:|:--------:|:--------:|
| **🟢 一级** | 金最优(全部) | 100.0% | 76,104 | ✅ 零波动 | ✅ 无失败 | ✅ 无 |
| **🟢 一级** | 金+升排+非孕+盈高 | 100.0% | 14,916 | ✅ 零波动 | ✅ 无失败 | ✅ 无 |
| **🟡 二级** | **金+升排+非孕** | **77.4%** | **468,801** | ⚠️ 波动大(±29.2pp) | ✅ 最差5% ≤0.09% | ⚠️ **显著下降** (95.2%→67.8%) |
| **🟡 二级** | **银+盈提示有** | **87.6%** | **66,191** | ⚠️ 波动±18.0pp | ⚠️ 最差5% ≤-3.12% | ✅ 近年稳定(86~87%) |
| **🔴 三级** | 金+升排 | 43.3% | 146,706 | ❌ 波动±26.6pp | ⚠️ -0.50% | 略降 |
| **🔴 三级** | 金+多长 | 46.9% | 747,867 | ❌ 波动±25.8pp | ⚠️ -0.81% | 略降 |
| **🔴 三级** | 银+龙猪 | 53.0% | 31,232 | ❌ 波动±25.8pp | ⚠️ -0.52% | 不稳 |
| **🔴 三级** | 银+柱排=升 | 70.2% | 150,051 | ❌ 波动±22.7pp | ⚠️ -0.00% | 近年略降 |
| **🔴 三级** | 银+ZA5~10 | 49.3% | 18,290 | ❌ 波动±25.7pp | ⚠️ -0.27% | 不稳 |
| **🔴 三级** | 银+WXAB=己 | 75.5% | 6,859 | ❌ 波动±20.8pp | ✅ 最差5% ≤0.00% | 不稳 |

### 关键发现

1. **仅一级信号可无条件信任。** 金最优和金+盈高 30 年零失败，P(≥3%)=100%，是所有策略的黄金标准。

2. **金+升排+非孕（二级）存在显著时间衰减。** P(≥3%) 从 2007 年巅峰 95.2% → 2025 年 67.8%，下降近 30pp。这个策略可用，但必须叠加额外条件（如标的类别、盈提示等）才能保持胜率。

3. **银+盈提示有（二级）是唯一稳定的银系策略。** 近年稳定在 86~87%，且样本量大(66,191周)，尾部风险控制在 -3.12% 以内，适合作为金系策略的补充。

4. **三级信号全部不稳定。** 年际波动普遍超过 20pp，不同市场环境下胜率差异巨大。仅适合作为参考信号，不应作为主要操作依据。

5. **尾部风险整体可控。** 除银+盈提示有外，所有策略的失败周最差5% 都在 0% 附近，说明失败周的跌幅温和，不会出现单次巨亏。

## 第一章 金条件策略

## 1.1 设计哲学 + 梯度分析

### 核心问题
> 是否只使用最优条件进行下周冲高判定？次要条件是否有分数？

### 答案

**周冲评分方案：概率即分数。** 策分 = 下周冲高≥3%的概率（P(≥3%)），直接查概率表赋值。

例如 "金+多长+升排+非孕" 在中证500上 P(≥3%)=90.3%，策分就是 90.3 分。
详见 [§1.2 概率即分数](#12-周冲评分方案概率即分数当前实现) 和 [§2.10 全量条件概率表](#210-全量条件概率表)。

**哲学：** 策分本质是 **0~100 的连续概率值**，不是"通过/不通过"的二元判定。全量梯度分析证实：

| 概率区间 | 周数 | 真均幅 | 相对基准 | 操作建议 |
|:------|:----:|:-----:|:--------:|:--------|
| 🏆 85~100 | 254272 | **11.90%** | 2.5x | A级 核心买入 |
| 🟢 70~84 | 648007 | **6.32%** | 1.33x | B级 辅助买入 |
| 🟡 50~69 | 638106 | **5.70%** | 1.20x | C级 观察 |
| 🟠 30~49 | 1703450 | **3.80%** | 0.80x | D级 回避 |
| 🔴 0~29 | 983395 | **2.95%** | 0.62x | E级 剔除 |
| **基准** | **4229602** | **4.76%** | 1x | — |

**关键结论：梯度平滑，无阈值效应。** 70~84 仍有 6.32% 真均幅（高出基准 32%），次优条件有价值。策分的连续分布保证了自然排序——高分高概率，低分低概率，无需人为分档。

### 次要条件的自然处理
概率即分数的设计天然处理了次要条件：条件越严格 → P(≥3%) 越高 → 策分越高，无需额外加权求和逻辑。

例如：
- 金+多长 → 中证500上 69.5分
- +升排 → 81.4分（+11.9pp）
- +非孕 → 90.3分（+8.9pp）
- +盈高+龙猪 → 100分（+9.7pp）

分数 = 历史数据说话，不是拍脑袋定的权重。

## 1.2 概率即分数 — 评分设计

### 1.2.1 核心概念

**策分 = 条件组合在下周冲高≥3%的历史概率。**

```
策分不再是"专家打分"，而是"历史数据说话"。

金+多长+升排+非孕 在中证500上 → 历史上H R≥3%的概率 = 90.3%
                                          ↓
                               策分 = 90.3分
```

**分数含义就是概率：** 策分 90 意味着"如果今日满足此条件，下周有 90% 的概率冲到 3% 以上"。每10次出现，就有9次达标。

### 1.2.2 为什么选 P(≥3%)？

| 计分基准 | 基准值 | 最优条件值 | 区分度 | 缺陷 |
|:---------|:-----:|:---------:|:-----:|:------|
| **P(>0)** | 91.5% | 100% | 8.5pp | 区分度太小，大部分组合都接近100% |
| **P(≥3%)** ✅ | **50.6%** | **100%** | **49.4pp** | **区分度最大，核心操作阈值** |
| P(≥5%) | 32.8% | 100% | 67.2pp | 最优条件以下大部分组合低于50%，参考性差 |

**P(≥3%) 是最佳平衡点：**
- 基准值 50.6% 恰好在中间，留出上下各 50pp 的分布空间
- 3% 是实际交易中有意义的冲高幅度（覆盖手续费+利润空间）
- 低于基准 50.6% 的条件自动表明"不如随机持有" → 不参与

### 1.2.3 分数意味着什么

| 策分 | 概率语言 | 操作语言 |
|:---:|:---------|:---------|
| 100 | 下周必冲到3%以上 | 最强信号，几乎无风险 |
| 90 | 10周中9周达标 | 极大概率，放心操作 |
| 80 | 10周中8周达标 | 高概率，正常操作 |
| 70 | 10周中7周达标 | 中高概率，可参与 |
| 60 | 10周中6周达标 | 中性，略优于基准 |
| 50 | 等于全量基准 | 无区分度，不操作 |
| <50 | 低于全量基准 | 不如不操作 |

### 1.2.4 VBA实现链路

```
神谕主循环:
  ① 条件判定 → 匹配策略名
     例: "金+多长+升排+非孕"

  ② 获取市板
     例: "Qic" (中证500)

  ③ 查概率表 P(≥3%)
     键 = "金+多长+升排+非孕|Qic" → 90.3

  ④ 写入
     谕组(X, 位谕of周冲策分) = 90.3
     谕组(X, 位谕of周冲策略) = "金+多长+升排+非孕"
```

源数据是 **全量7463只×4229602周回测**，按条件+市板组合统计 HR≥3% 的频率，存入 `D:\zdata\照明概率.txt`。VBA 首次调用时加载到 Dictionary 缓存，后续查表为 O(1)。

### 1.2.5 概率表的结构

概率表是一个 **策略×市板 的矩阵**，每格 = P(≥3%)：

```

金+多长+升排+非孕  81.6  50.0  48.5  86.6  90.3   91.4  90.5
银+盈提示有       83.3  59.6  63.8  85.2  88.2   87.2  86.4
```

- **行 = 策略名**（共17个条件策略，从"全量基准"到"龙猪/管+盈提示有"）

## 1.8 策分卡七大结论 + 核心结论

### 1. 体积效应：市值越小冲高越大
中证2000(5.39%) > 中证1000(5.22%) > 中证500(4.96%) > 沪深300(4.52%) > ETF(2.45%)
- 小盘股波动大，冲高机会多，但风险也大
- 大盘蓝筹冲高中等但稳定
- ETF/指数只适合做基准仓位，不适合冲高策略

### 2. 真冲高率92% = 天然优势
任何股票在任何一周都有92%概率盘中冲高。这不是策略的功劳，是市场规律。
**策略的核心价值不是提高冲高率，而是筛选冲高幅度更大的标的。**

### 3. 多长区是唯一有效操作区
- 四域=多长时真均幅+4.32%（全量），远优于多被(+3.46%)
- 多长区内的震仓频率决定月基分高低

### 核心结论摘要

1. **真冲高基准91.5%**，最优条件下达100%。策略核心价值不在提高冲高概率，在**筛选冲高幅度**。
2. **最具操作价值**的是（金+升排+非孕）：81.0%概率冲到3%以上，**每10周就有8周达标**。

### 2.9 概率即分数（已实现）

**核心思路：** 不再"给条件赋分"，而是**用条件自身的下周冲高概率作为分数**。

```
传统方案：四域=多长 → 拍脑袋给20分
概率方案：四域=多长 → 下周冲高概率95.5% → 分数就是95.5
```

**优势：**
- 分数天然可解释（"策分96" = "下周有96%概率冲高"）
- 不需要设计权重，概率就是权重
- 每个条件组合的分数随市场自动变化（需定期更新概率表）
- 基础条件以下自动得0分（例如全量基准91.5%，低于此的都不参与）

**VBA输出两个变量：**

```
输出结构：
  策略名称: "金+甲乙己+升排+非孕"
  冲高概率: 99.7%
```

**按类别细分的必要性：**
同一个条件在不同类别基金中的下周冲高概率不同。例如"银+盈提示有"在沪深300和中证2000中的概率可能相差10个百分点。因此需要按类别生成概率表，VBA查表赋值。

#### 2.1.4 对VBA代码的影响

当前 `IQQQ跨码_D据擎2神谕.bas` 中`_多长`区旧逻辑将基仓/浮仓混在同一个区域内，需重构为独立分层：

```
旧逻辑：
  _多长区 → 基仓(BTZA>0) / 浮仓(BTZA≤0) / 减仓(BTZC≤0) / 清仓(BTZE≤0)

新逻辑：
  _多长区 → 仅基仓（单一仓位，满足条件即持有，不下破DJE不退）
  浮仓   → 独立操作模式，由冲高信号触发，不隶属_多长区
              ├── <周冲策略>：基于周柱排/周波型，DXZA<0或DXAB<0时启用
              └── <日冲策略>：基于日柱排，DXAB>0且DXZA>0时启用

  仓位跟随区划与信号：
    _多长      → 持有基仓
    _多被   → 可参与（视情况开基仓）
    _空看    → 不参与
    _空长   → 不参与
    浮仓信号触发 → 在_多长保护下开浮仓
```

**分层策略决策树（VBA实现）：**

```
                  ┌─ <月基策略> ──→ 长期策略（针对_多长，按WXZC>0持有）
                  │
当前持仓 ──────────┤
                  │          ┌─ 能<日冲策略>？→ <日冲策略>策略
                  └─ 浮仓 ──┤
                            └─ 不能？→ 能<周冲策略>？→ <周冲策略>策略
                                      │
                                      └─ 都不能？→ 平浮仓，回归基仓
```
