## 二、核心数据结构

### 2.1 谕组 (Variant 2D Array) — 核心数据契约

| 维度 | 说明 |
|------|------|
| 行 | 每行 = 一支股票 |
| 列 | 150+ 列指标，从 `位谕列始全部` (位qt主宽+1) 到 `位谕列终全部` |
| 族 | 周段龄 → 周波 → 周龟 → 周管 → 周层 → 日层 → 日管 → 日波 → 日丘 → 日波龄 → 日段 → 日基 → 日德 → 仓 → 周基 → 周德 → 周道 → 周奏 → 月类 → 日均 |

**关键字段：**
- `位谕of仓周类` (= 位谕始of族仓 + 15) — 仓周类编码
- `位谕of仓日类` (= 位谕始of族仓 + 16) — 仓日类编码
- `位谕of周层三鳄` — 核心分类指标
- `位谕of日层联动` — 日周联动指标

### 2.2 OS列 (位os始柱波类 ~ 位os终统高类) — 操盘归集

| 子族 | 列数 | 说明 |
|------|------|------|
| 柱波类 | 27列 | 结价/前价/临价 (收开高低量换比幅) |
| 柱组类 | 79列 | PR0-9/HR0-9/LR0-9/H0-9/L0-9/O0-9/C0-9/BTPR/BTHR |
| 均线类 | 38列 | JZ/JA/JB/JC/JD/JE/JF/JG + BT交叉 |
| 奏类 | 21列 | 均线交叉统计 |
| 阶部类 | 15列 | 局系统 |
| 龟类 | 28列 | 顶/底/哼/哈/叠幅/极/态/触 |
| 具类 | 28列 | 上/中/并符串 + 宽/幅符串 |
| 算法类 | 18列 | 累持/累赚/层界/层护 |
| 基类 | 3列 | 代称/市期/累龄 |
| 统涨类 | 30+列 | 统计涨幅/线类/区类/高类 |

### 2.3 QT列 (位qt代码 ~ 位qt主宽=27) — 全息概览

| 列 | 说明 |
|----|------|
| 1-14 | 代码/代称/涨幅/高幅/收/前收/开/高/低/量/量比/换手/额/市值 |
| 15-17 | 停牌/今时/今期 |
| 18-26 | 成分/市板/行业/行益/概念/仓主/仓周/今池/在池 |
| 27 | 盯盘类别 |

## 四、核心调用链

### 4.1 每日结算流程

```
STBASE结算管理_全新结算调程花天()
  └─> STBASE结算引擎_全新结算正程(花册, 文件夹, 日期范围)
       ├─> STBASE结算引擎_全新结算数程由外库(组结算, 花册WS)
       │    └─> 逐票: STBASE结算引擎_单点结算核程(ARROS)
       │         └─> STBASE结算引擎_单点初始核程()
       │         └─> STBASE结算引擎_区间基码/基组数程()
       │         └─> STBASE结算引擎_全点基组数程()
       ├─> STBASE结算引擎_历统时正程()
       │    └─> 历研统码基程_展日码总()
       │    └─> 历研统时引擎1主程()
       └─> 写花册 → 格式化
```

### 4.2 全息→神谕→筛选→瓜页

```
IQQQ跨码据擎_数程生成全息(谕组, 花册, 实结类型)
  ├─> 读取花册 → 采样/过滤
  ├─> 复制QT列基础信息
  ├─> 查询行情(UBHTTP腾实_查回典批量)
  └─> IQQQ跨码据擎_数程生成跨时神谕(谕组, 组结算)
       └─> 逐行计算: 周段龄→周波→周龟→周管→周层→日层→日管→日波→日丘→日段→日基→日德→仓→月类→日均

IQQQ展擎筛程至C0章/至D0节/至A1册/至A2册/至A9册/至B1书
  └─> 遍历谕组 → 按仓周类/仓日类/市板/股性/池分类 → 输出到对应Sheet

IQQQ展擎瓜页引擎基程(WB, 指定表名, 元组本页, 瓜分类型)
  ├─> 更新模式: IQQQ展擎筛程_出程更新通用()
  │    └─> 重新生成谕组 → 写表 → 格式化 → 排序
  └─> 瓜分模式: IQQQ展擎筛程至C0章/至D0节/至D1节/至D3节/至D4节
       └─> 按仓周类/仓日类/三鳄分类 → 输出到不同章节
```

### 4.3 定时刷新

```
UGSOP定时引擎开始()
  └─> 每5秒回调: UGSOP定时引擎刷新持页()
       └─> IQQQ展擎瓜页前调更新排日sSCC()
            └─> IQQQ展擎瓜页引擎基程() → 更新模式 → 重新生成谕组
```

## 五、脑暴/思考

### 5.1 系统设计哲学

1. **三层分类体系**
   - 仓周类: 金/银/唏/嘘/尿/屎 × abcryz (周粒度)
   - 仓日类: 上/忐/忠/忑/中/下/无 × abcryz (日粒度)
   - 三鳄: 周层三鳄 (核心分类指标)

2. **日周联动**
   - 周线决定方向和止损 (WJC/WJB)
   - 日线决定入场和止盈 (DJC/DJE)
   - 仓周类/仓日类通过位谕of日层联动连接

3. **多市场统一框架**
   - 中股/美股/港股通/港股指/中期 → 统一花册结构
   - 统一结算引擎 → 不同数据源适配

### 5.2 可优化点
